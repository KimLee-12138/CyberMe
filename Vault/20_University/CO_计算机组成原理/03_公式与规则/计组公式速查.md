# 计组公式速查

## 一、性能

- CPU时间 = 指令数 × CPI × 时钟周期 = (指令数 × CPI) / f
- CPI = Σ(CPIi × 比例i)
- MIPS = f / (CPI × 10^6)

## 二、Amdahl 定律

Speedup = 1 / ((1-f) + f/S)

## 三、Cache

- 平均访存时间 = 命中时间 + 缺失率 × 缺失代价
- 地址 = Tag | Index | Offset
- Cache 行数 = 容量 / (块大小 × 相联度)

## 四、浮点数(IEEE 754)

单精度 Bias=127：N = (-1)^S × 1.M × 2^(E-127)
双精度 Bias=1023：N = (-1)^S × 1.M × 2^(E-1023)

## 五、流水线

加速比 ≈ 流水线段数 (理想)
吞吐率 = 指令数 / 总时间

## 六、补码

[X]补 = 2^n + X (mod 2^n)
-128的8位补码 = 1000 0000(没有对应正数)

## 七、存储容量

n 位地址总线 → 寻址能力 = 2^n 单元
存储容量 = 2^地址线数 × 数据线位宽
