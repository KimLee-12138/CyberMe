# 数据结构总览

> 来源：DataStructure(1).pdf（作者 LI LIANGJI，99页完整讲义）

## 知识结构

```text
数据结构
├─ 线性结构
│  ├─ 线性表（顺序表 / 单链表 / 循环链表 / 双向链表）
│  ├─ 栈（顺序栈 / 链栈）→ LIFO
│  ├─ 队列（循环队列 / 链队）→ FIFO
│  └─ 字符串（BF / KMP 匹配）
│
├─ 树形结构
│  ├─ 二叉树（遍历 / BST / AVL）
│  ├─ 树与森林（↔ 二叉树转换）
│  └─ 哈夫曼树（最优二叉树，编码压缩）
│
├─ 图形结构
│  ├─ 存储（邻接矩阵 / 邻接表）
│  ├─ 遍历（DFS / BFS）
│  ├─ 最小生成树（Prim / Kruskal）
│  └─ 最短路径（Dijkstra / Floyd）
│
└─ 算法应用
   ├─ 查找（顺序 / 二分 / 哈希 / BST / B树）
   └─ 排序（冒泡/插入/选择 → 归并/快排/堆 → 基数）
```
