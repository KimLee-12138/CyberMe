# 树与二叉树

## 一句话解释
二叉树每个节点最多两个子节点。遍历有四种（前序/中序/后序/层序），每种都有递归和非递归实现。二叉搜索树(BST)左<根<右，平均O(log n)；AVL树严格平衡(|平衡因子|≤1)；哈夫曼树是最优二叉树(WPL最小)。

## 核心原理
**二叉树性质(5条必须记住)**：
1. 第i层最多2^{i-1}个节点(i≥1)
2. 深度为k的二叉树最多2^k-1个节点
3. **n₀=n₂+1**（叶子数=度为2的节点数+1）——考试必考
4. n个节点的完全二叉树深度=⌊log₂n⌋+1
5. 完全二叉树编号i：左孩子2i，右孩子2i+1，父节点⌊i/2⌋

**四种遍历**：
| 遍历 | 顺序 | 递归模板 | 应用 |
|------|------|---------|------|
| 前序 | 根→左→右 | visit(root); pre(L); pre(R) | 复制树、前缀表达式 |
| 中序 | 左→根→右 | in(L); visit; in(R) | **BST中序=有序序列！** |
| 后序 | 左→右→根 | post(L); post(R); visit | 销毁树(先删孩子) |
| 层序 | 上→下，左→右 | 队列+BFS | 最短路径(树中) |

**非递归遍历**：用栈模拟递归。前序简单(根入栈→弹出→右左孩子入栈)；中序需要一路向左压栈。

**线索二叉树**：利用空指针域(n个节点有n+1个空指针)存储前驱/后继。中序线索化最常用。

**二叉搜索树BST**：左子树所有值<根<右子树所有值。
- 查找/插入：从根开始比较，小于去左大于去右。O(h)。
- 删除：(1)叶子直接删 (2)单孩子→孩子替代 (3)两个孩子→用**中序后继**(右子树最小)或**中序前驱**(左子树最大)替代
- BST可能退化(有序插入→链状)，O(n)。AVL解决这个问题。

**AVL树**：|平衡因子|=|左高-右高|≤1。插入后不平衡→四种旋转：
- LL：右旋一次 | RR：左旋一次 | LR：左旋再右旋 | RL：右旋再左旋

**树/森林↔二叉树**：左孩子右兄弟表示法。
- 树→二叉树：每个节点的第一个孩子作为左孩子，其余兄弟依次作为右孩子
- 二叉树→森林：二叉树的根及其左子树=第一棵树，根的右子树=剩余森林

## 完整例子
```cpp
void inorder(TreeNode* root) { // 中序递归
    if(!root) return;
    inorder(root->left);
    visit(root);
    inorder(root->right);
}
int depth(TreeNode* root) { // 求深度
    if(!root) return 0;
    return 1+max(depth(root->left), depth(root->right));
}
```

## 容易出错的地方
- 前/中/后序的**访问时机**不同。前序=第一次遇到节点就访问，中序=从左子树返回时访问，后序=从右子树返回时访问
- BST的删除(两个孩子)用中序后继替代，不能随便选
- AVL旋转方向判断：看插入发生在"发现不平衡节点"的哪一侧的子树的哪一侧
- 树转二叉树后，原树的兄弟关系变为二叉树的父子(右孩子)关系
