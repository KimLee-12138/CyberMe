# 二叉树与树 — 例题集

## 例题1：二叉树性质

**题目**：一棵完全二叉树有 1000 个节点，求叶子节点数。

**解题步骤**：
1. 完全二叉树中，n₀ = n₂ + 1，n = n₀ + n₁ + n₂
2. n₁ 只能为 0 或 1（完全二叉树性质）
3. n₀ + n₁ + n₂ = 1000，代入 n₂ = n₀ - 1
4. n₀ + n₁ + (n₀ - 1) = 1000 → 2n₀ + n₁ = 1001
5. 若 n₁ = 0：2n₀ = 1001 → n₀ = 500.5（不合法）
6. 若 n₁ = 1：2n₀ = 1000 → n₀ = 500

**答案**：叶子节点数 = 500

**易错点**：完全二叉树的 n₁ 只能是 0 或 1，要分别验证。

## 例题2：遍历序列推导

**题目**：二叉树前序序列为 ABDEGCFH，中序序列为 DBGEACHF，画出二叉树并写后序序列。

**解题步骤**：
1. 前序第一个 A 是根节点
2. 在中序中找到 A，左边 DBGE 是左子树，右边 CHF 是右子树
3. 左子树前序是 BDEG，中序是 DBGE → B 为左子树根
4. 在中序中 B 左边 D，右边 GE → D 为 B 的左孩子
5. GE 前序为 EG，中序为 GE → E 为根，G 为左孩子
6. 右子树前序为 CFH，中序为 CHF → C 为根
7. 中序中 C 左边空，右边 HF → H 为 C 左孩子，F 为 H 右孩子（前序 FH）

**答案**：
```
        A
      /   \
     B     C
    / \   /
   D   E H
      /   \
     G     F
```
后序序列：D G E B H F C A

**易错点**：中序中根左边的全是左子树，右边的全是右子树，这个规则不能搞反。

## 例题3：哈夫曼树 WPL

**题目**：字符频率 {a:7, b:5, c:2, d:4, e:9}，构造哈夫曼树并求 WPL。

**解题步骤**：
1. 选最小的两个：c(2) + d(4) = 6
2. 现在有 {a:7, b:5, e:9, (c+d):6}，选 b(5) + (c+d)(6) = 11
3. 现在有 {a:7, e:9, (b+c+d):11}，选 a(7) + e(9) = 16
4. 现在有 {(b+c+d):11, (a+e):16}，合并 = 27
5. WPL = 2×3 + 4×3 + 5×3 + 7×2 + 9×2 = 6 + 12 + 15 + 14 + 18 = 65

**答案**：WPL = 65

**验证**：所有叶节点权值 × 到根的边数求和。c 和 d 最深（3 条边），a 和 e 最浅（2 条边）。

**易错点**：WPL 是**叶子节点**的权值 × 路径长度，不是所有节点。

## 例题4：树转二叉树

**题目**：树有 3 个孩子节点的节点数 2 个，有 2 个孩子 3 个，有 1 个孩子 4 个，叶子数未知。转为二叉树后，**右子树为空的节点**有几个？

**解题步骤**：
1. 树转二叉树使用「左孩子右兄弟」表示法
2. 二叉树中，节点的右指针指向原树中的**下一个兄弟**
3. 在原树中**没有右兄弟**的节点 → 二叉树中右子树为空
4. 哪些节点在原树中没有右兄弟？
   - 每个父节点的**最后一个孩子**
   - 根节点（没有兄弟）
5. 有 2 个孩子的节点：最后一个孩子 = 2 个，有 3 个孩子：最后一个孩子 = 2 个
6. 有 1 个孩子的节点：该孩子也是最后一个 = 4 个
7. 叶子也是其父的「某个孩子」，但问题问的是右子树为空的节点，叶子在原树中也可能有右兄弟
8. 综合：根(1) + 每父节点的最后一个孩子(2+3+4) = 10

**答案**：10 个节点右子树为空

**易错点**：根节点也没有兄弟，所以也要算进去。
