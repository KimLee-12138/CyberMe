# 例题_线性代数

## 例题1：行列式计算

**题目**：计算 |1 2 3; 4 5 6; 7 8 9|。

**答案**：行列式=0（第三行=第一行×2-第二行，线性相关→秩<3→行列式为0）。

## 例题2：矩阵求逆

**题目**：求 A=[1 2; 3 4] 的逆矩阵。

**答案**：|A|=-2。A⁻¹=(-1/2)[4 -2; -3 1]=[-2 1; 1.5 -0.5]。验证：A×A⁻¹=I。

## 例题3：特征值

**题目**：求 A=[3 1; 0 2] 的特征值和特征向量。

**答案**：|A-λI|=(3-λ)(2-λ)=0→λ₁=3,λ₂=2。λ₁=3：(A-3I)x=0→[0 1; 0 -1]x=0→x₁=t[1;0]。λ₂=2：x₂=s[1;-1]。

## 例题4：对角化

**题目**：判断 A=[2 0; 0 2] 是否可对角化。写出对角化形式。

**答案**：可。λ₁=λ₂=2（重根但有两个线性无关的向量[1;0]和[0;1]）。A=I·(2I)·I⁻¹，已经是对角阵。
