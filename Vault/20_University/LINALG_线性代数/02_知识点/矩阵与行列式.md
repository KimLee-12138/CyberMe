# 矩阵与行列式

## 一句话解释

矩阵是数字的矩形阵列，行列式是一个标量。|A|≠0 等价于 A 可逆、Ax=b 有唯一解、A 的列（行）线性无关。秩是矩阵的核心不变量——rank(A) = 线性无关的最大列数 = 线性无关的最大行数 = 非零子式的最大阶数。

## 核心原理

### 矩阵运算

**加法**：A+B = [aᵢⱼ+bᵢⱼ]（同型矩阵才能相加）

**数乘**：kA = [k·aᵢⱼ]

**乘法**：A(m×n)·B(n×p) = C(m×p)，Cᵢⱼ = Σₖ₌₁ⁿ Aᵢₖ·Bₖⱼ

矩阵乘法的核心理解：**C 的第 j 列 = A 的各列以 B 的第 j 列为系数的线性组合**。

**性质**：
- 结合律：(AB)C = A(BC) ✓
- 分配律：A(B+C) = AB+AC ✓
- **交换律不成立**：AB ≠ BA ✗
- (AB)^T = B^T A^T（逆序规则）
- (AB)⁻¹ = B⁻¹A⁻¹（逆序规则）

### 行列式

**2×2**：|a b; c d| = ad - bc

**3×3**（Sarrus法则）：正对角线乘积和 - 反对角线乘积和

**n×n 的计算**：

1. **按行/列展开（Laplace 展开）**：
|A| = Σ_{j=1}^n (-1)^{i+j}·aᵢⱼ·Mᵢⱼ
其中 Mᵢⱼ 是去掉第 i 行第 j 列后的子式（代数余子式 Aᵢⱼ = (-1)^{i+j}Mᵢⱼ）

2. **三角化法**（最实用）：
用行变换化为上三角，|A|=对角线乘积。（行交换会变号！行倍乘会乘倍！）

**行列式性质**：
- |A^T| = |A|
- |AB| = |A|·|B|
- |A⁻¹| = 1/|A|
- 交换两行（列）→ 行列式变号
- 某行乘 k → 行列式乘 k
- 某行加另一行的倍数 → 行列式不变

### 逆矩阵

A·A⁻¹ = A⁻¹·A = I

**伴随矩阵法**：A⁻¹ = (1/|A|)·adj(A) = (1/|A|)·[Aᵢⱼ]^T
其中 Aᵢⱼ 是代数余子式。

**初等变换法**：[A|I] → 行变换 → [I|A⁻¹]

### 矩阵的秩

**秩** = 矩阵中非零子式的最大阶数 = 行秩 = 列秩
= 行阶梯形中主元（pivot）的个数

**秩定理**：rank(A) + nullity(A) = n（A 的列数）

**秩的性质**：
- rank(AB) ≤ min(rank(A), rank(B))
- rank(A+B) ≤ rank(A) + rank(B)
- A 可逆 ⇔ rank(A) = n（满秩）

### 线性方程组

Ax = b

**克拉默法则**（|A|≠0 时）：xᵢ = |Aᵢ|/|A|，Aᵢ 是用 b 替换 A 第 i 列得到的矩阵。

**解的结构**：Ax=b 的通解 = Ax=b 的一个特解 + Ax=0 的通解（齐次解）

**有解判定**：rank(A) = rank([A|b])（系数矩阵的秩 = 增广矩阵的秩）

## 前置知识

- 向量的基本运算（加法、数乘）
- 高斯消元法

## 完整例子

**2×2 求逆**：A = [2 1; 1 3]
- |A| = 6-1 = 5
- adj(A) = [3 -1; -1 2]（对角线交换，副对角线变号）
- A⁻¹ = (1/5)[3 -1; -1 2]

**解方程**：x+2y=5, 3x+4y=6
- 克拉默法则：|A| = 1×4-2×3 = -2
- x = |5 2; 6 4|/(-2) = (20-12)/(-2) = -4
- y = |1 5; 3 6|/(-2) = (6-15)/(-2) = 4.5

## 容易出错的地方

- AB≠BA！不要习惯性地认为乘法可交换
- 行列式|A+B|≠|A|+|B|（没有加法分配律）
- 课程导航：[[../00_线性代数_课程MOC|线性代数 MOC]]


- (AB)⁻¹=B⁻¹A⁻¹，是**逆序**不是顺序！
- 克拉默法则只在|A|≠0时能用
- 秩不是独立列的数量——是"线性无关列的最大数量"
