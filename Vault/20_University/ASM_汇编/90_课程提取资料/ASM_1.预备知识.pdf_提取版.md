## 第1页

1.预备知识   
1. 汇编语言：汇编指令是机器指令的助记符，和机器指令一一对应。汇编语言由汇编指令，伪指令，
其他符号组成
程序员 -> 汇编指令 -> 编译器 -> 机器指令 -> 计算机
1. 寄存器（如 ax ， bx ）： CPU 中可以存储数据的器件，一个 CPU 有多个寄存器
2. 存储器：向 CPU 提供指令和数据。指令和数据都存放在存储器中，也就是内存之中
3. CPU对存储器的读写：
存储单元的地址（地址信息）
器件的选择（控制信息）
读或写的数据（数据信息）
总线（ CPU 通过总线将这三类信息传送到存储芯片中）：
种类：地址总线、数据总线、控制总线
作用：
CPU通过地址总线指定存储单元的地址
CPU通过数据总线和内存或其他器件之间进行数据传送
CPU通过控制总线传送对外部器件的控制信息
不同的总线如何影响 CPU 的性能：
地址总线能传送多少不同信息， CPU 就能对多少内存单元进行寻址，地址总线宽度为 N 时， CPU 最
多可以寻找 2 的 N 次方个内存单元。地址总线宽度越宽， CPU 寻址性能越好
数据总线的宽度决定了 CPU 和外界的数据传送的速度。有多少根数据总线， CPU 就能一次性传送多
少位的数据。
控制总线的宽度决定了 CPU 对外部器件的控制能力。控制总线越多， CPU 对其他器件的控制能力越
强
1. 一个存储单元可以存储 8bit 即一个字节 1B
2. 主板和接口卡：主板相当于一个容器，主板上放有核心器件以及其他器件（如 CPU 、存储器等等）
各类存储器芯片
从读写功能上分类：  RAM （随机存储器）  ROM （只读存储器）
内存地址空间

## 第2页

最终运行程序的是 CPU 。我们用汇编编程的时候，必须从 CPU 角度考虑问题。对 CPU 来讲，系统中
的所有存储器的存储单元都处于一个统一的逻辑存储器中，它的容量受 CPU 寻址能力的限制。这个
逻辑存储器即是我们所说的内存地址空间
