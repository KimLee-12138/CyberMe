# 8086 常用指令

> 一句话：8086 指令分六大类——数据传送、算术运算、逻辑运算、转移控制、栈操作、串操作。

## 一、数据传送指令

### 1.1 MOV

```asm
mov 目的, 源    ; 源→目的
```

**限制**：
- 不能 `mov cs, ax`（不能修改 CS）
- 不能 `mov ip, ax`（不能修改 IP）
- 不能 `mov [bx], [si]`（不能内存→内存）
- 段寄存器不能做目的操作数（`mov ds, 1234H` 也不行——需通过 AX 中转）

**段寄存器赋值**：
```asm
mov ax, 1234H   ; 通过 AX 中转
mov ds, ax       ; DS = 1234H
```

### 1.2 XCHG

```asm
xchg ax, bx     ; 交换 AX 和 BX
```

交换两个操作数的内容，至少有一个是寄存器。

### 1.3 LEA（取偏移地址）

```asm
lea bx, [si+4]   ; BX = SI + 4（取地址，不是取值）
```

## 二、算术运算指令

| 指令 | 格式 | 说明 | 影响标志位 |
|------|------|------|-----------|
| ADD | `add ax, bx` | AX=AX+BX | CF,OF,ZF,SF,PF |
| ADC | `adc ax, bx` | AX=AX+BX+CF（带进位加法） | CF,OF,ZF,SF,PF |
| SUB | `sub ax, bx` | AX=AX-BX | CF,OF,ZF,SF,PF |
| SBB | `sbb ax, bx` | AX=AX-BX-CF（带借位减法） | CF,OF,ZF,SF,PF |
| INC | `inc ax` | AX++ | OF,ZF,SF,PF（不影响CF！） |
| DEC | `dec ax` | AX-- | OF,ZF,SF,PF（不影响CF！） |
| MUL | `mul bl` | 无符号乘法 | CF,OF |
| IMUL | `imul bl` | 有符号乘法 | CF,OF |
| DIV | `div bl` | 无符号除法 | 不定义 |
| IDIV | `idiv bl` | 有符号除法 | 不定义 |

### 2.1 乘除法细节

**乘法**——隐含操作数：

| 操作数大小 | 乘数 | 被乘数(隐含) | 结果 |
|-----------|------|-------------|------|
| 8位 | `mul bl` | AL | AX = AL × src |
| 16位 | `mul bx` | AX | DX:AX = AX × src |

**除法**——隐含操作数：

| 操作数大小 | 除数 | 被除数(隐含) | 商 | 余数 |
|-----------|------|-------------|----|------|
| 8位 | `div bl` | AX | AL | AH |
| 16位 | `div bx` | DX:AX | AX | DX |

**注意**：除法前必须确保被除数足够宽（8位除要扩展AX，16位除要扩展DX:AX）。

## 三、逻辑运算与移位

| 指令 | 说明 | 典型用途 |
|------|------|---------|
| AND | 按位与 | 置 0 特定位：`and al, 11011111B`（第5位置0） |
| OR | 按位或 | 置 1 特定位：`or al, 00100000B`（第5位置1） |
| XOR | 按位异或 | 清零：`xor ax, ax`（比 mov ax,0 快） |
| NOT | 按位取反 | — |
| SHL | 逻辑左移 | 乘 $2^n$ |
| SHR | 逻辑右移 | 无符号除 $2^n$ |
| SAR | 算术右移 | 有符号除 $2^n$（保持符号位） |

## 四、转移控制指令

### 4.1 无条件转移

```asm
jmp short 标号       ; 段内短转移 (-128~127)
jmp near ptr 标号    ; 段内近转移 (-32768~32767)
jmp far ptr 标号     ; 段间远转移 (CS和IP都改)
jmp ax              ; IP = AX（寄存器跳转）
```

### 4.2 条件转移（配合 CMP）

```asm
cmp ax, bx          ; 计算 AX-BX，不保存结果，只改标志位
je 标号             ; ZF=1（相等）→ 跳转
jne 标号            ; ZF=0（不等）→ 跳转
ja 标号             ; 无符号 大于 → 跳转
jb 标号             ; 无符号 小于 → 跳转
jg 标号             ; 有符号 大于 → 跳转
jl 标号             ; 有符号 小于 → 跳转
jcxz 标号           ; CX=0 → 跳转
```

**jxx 是段内短转移**：范围 -128 ~ +127 字节。

### 4.3 CALL 和 RET

```asm
call 子程序         ; push IP, jmp 子程序
ret                 ; pop IP（返回调用点）
```

**段间调用**：`call far ptr` → push CS, push IP, jmp far

## 五、栈操作

```asm
push ax             ; SP-=2, SS:[SP]=AX
pop ax              ; AX=SS:[SP], SP+=2
pushf               ; 标志寄存器入栈
popf                ; 栈顶弹出到标志寄存器
```

**栈增长方向**：向低地址（SP 减小）。

## 六、串操作

| 指令 | 说明 | 配合寄存器 |
|------|------|-----------|
| MOVSB | 传送字节：`ES:[DI] = DS:[SI]`，SI/DI ±1 | DF 控制方向 |
| MOVSW | 传送字：`ES:[DI] = DS:[SI]`，SI/DI ±2 | DF 控制方向 |
| STOSB | 存字节：`ES:[DI] = AL`，DI ±1 | — |
| LODSB | 取字节：`AL = DS:[SI]`，SI ±1 | — |
| REP | 重复前缀：`rep movsb` → CX 次 | CX 控制次数 |

**DF 方向标志**：
- `cld` → DF=0 → SI/DI **递增**（正向）
- `std` → DF=1 → SI/DI **递减**（反向）

## 七、易错点

1. **INC/DEC 不影响 CF**：用 `inc` 做循环计数时，不能靠 CF 判断
2. **`mov [bx], [si]` 非法**：8086 不支持内存到内存传送
3. **除法溢出**：被除数高位 > 除数 → 除法溢出（0 号中断）
4. **jxx 只能在 -128 ~ +127 内跳转**：超出范围用 `jmp near ptr`
