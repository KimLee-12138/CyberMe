# 8086 寄存器

> 一句话：8086 有 14 个 16 位寄存器。掌握每个寄存器的用途和隐含用法是写汇编的基础。

## 一、寄存器全览

```
8086 CPU (16位)
├─ 通用寄存器 (4个)
│  AX (AH|AL) — 累加器，乘除/I/O默认
│  BX (BH|BL) — 基址寄存器，可间接寻址
│  CX (CH|CL) — 计数器，loop默认
│  DX (DH|DL) — 数据寄存器，乘除/I/O辅助
├─ 变址寄存器 (2个)
│  SI — 源变址寄存器
│  DI — 目的变址寄存器
├─ 指针寄存器 (2个)
│  SP — 堆栈指针（指向栈顶）
│  BP — 基址指针（默认段SS）
├─ 段寄存器 (4个)
│  CS — 代码段
│  DS — 数据段
│  SS — 堆栈段
│  ES — 附加段
├─ 指令指针 (1个)
│  IP — 指令指针（不能直接读写！）
└─ 标志寄存器 (1个)
   PSW/FLAGS — 状态标志 + 控制标志
```

## 二、通用寄存器

| 寄存器 | 高8位 | 低8位 | 特殊用途 |
|--------|-------|-------|---------|
| AX | AH | AL | 乘除法默认操作数、I/O 指令数据口 |
| BX | BH | BL | 唯一可间接寻址的通用寄存器 |
| CX | CH | CL | `loop` 指令的默认计数器 |
| DX | DH | DL | 乘除法辅助（32位结果高16位）、I/O 端口地址 |

**拆分使用示例**：
```asm
mov ax, 1234H   ; AX = 1234H
; 此时 AH = 12H, AL = 34H
mov al, 56H     ; AX = 1256H (只改低8位)
```

## 三、变址与指针寄存器

| 寄存器 | 默认段 | 主要用途 |
|--------|--------|---------|
| SI | DS | 源操作数地址（串传送指令自动使用） |
| DI | ES | 目的操作数地址（串传送指令自动使用） |
| SP | SS | 栈顶指针（push/pop 自动修改） |
| BP | SS | 访问栈中数据和参数 |

## 四、段寄存器详解

### 4.1 CS 和 IP

CS:IP 指向 CPU **当前要读取**的指令地址。

**取指令的物理地址** = CS × 16 + IP

IP 的变化规律：
- 顺序执行：IP = IP + 当前指令字节数
- `jmp`：IP 被替换为目标地址/位移量
- `call`：当前 IP 压栈，IP 跳转到目标
- `ret`：从栈中弹出返回地址到 IP
- `int`：标志寄存器+CS+IP 依次压栈，IP 从中断向量表加载

### 4.2 DS

数据段寄存器。几乎所有内存访问默认以 DS 为段地址：

```asm
mov ax, [bx]       ; AX = DS:[BX]
mov ax, [1234H]    ; AX = DS:[1234H]
```

### 4.3 SS 和 SP

**任意时刻 SS:SP 指向栈顶元素**。

```
push ax:
  SP = SP - 2        (栈向低地址增长)
  SS:[SP] = AX        (将内容压入新栈顶)

pop ax:
  AX = SS:[SP]        (从栈顶取内容)
  SP = SP + 2          (栈顶指针回升)
```

**栈空时 SP = 0 → push 后 SP = FFFEH**（SP 从 0 减 2 绕回）。

## 五、标志寄存器 PSW (FLAGS)

8086 的 PSW 共 16 位，其中 9 位有效：

| 位 | 名称 | 说明 |
|----|------|------|
| 0 | CF | 进位/借位（无符号溢出） |
| 2 | PF | 奇偶标志（低8位1的个数偶数→PF=1） |
| 4 | AF | 辅助进位（BCD码用） |
| 6 | ZF | 零标志（结果=0→ZF=1） |
| 7 | SF | 符号标志（结果最高位） |
| 8 | TF | 单步调试标志 |
| 9 | IF | 中断允许标志 |
| 10 | DF | 方向标志（串传送方向） |
| 11 | OF | 溢出标志（有符号溢出） |

**CF vs OF**：
- CF 针对**无符号数**：255+1=0（进位），CF=1
- OF 针对**有符号数**：127+1=-128（溢出），OF=1

## 六、王爽教材中的寄存器规律

1. **AX-BX-CX-DX 都可以拆成两个 8 位寄存器**（H/L）
2. **只有 BX、SI、DI、BP 可以放在 `[]` 中做间接寻址**
3. **不能用 mov 修改 CS、IP**
4. **push/pop 操作数必须是 16 位**（不能 push al）
