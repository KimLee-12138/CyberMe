# 内存寻址

> 一句话：8086 提供 7 种寻址方式，核心是理解 `[寄存器组合]` 如何映射到物理地址。

## 一、寻址方式总览

| 寻址方式 | 格式 | 示例 | 说明 |
|---------|------|------|------|
| 立即寻址 | `mov ax, 1234H` | 数据在指令中 | 最快 |
| 寄存器寻址 | `mov ax, bx` | 数据在寄存器中 | CPU内部传递 |
| 直接寻址 | `mov ax, [1234H]` | 偏移地址直接给出 | DS:[1234H] |
| 寄存器间接寻址 | `mov ax, [bx]` | BX 存储偏移地址 | DS:[BX] |
| 基址寻址 | `mov ax, [bx+5]` | BX/BP + 位移量 | 结构体/数组 |
| 变址寻址 | `mov ax, [si+5]` | SI/DI + 位移量 | 数组遍历 |
| 基址变址寻址 | `mov ax, [bx+si]` | BX/BP + SI/DI | 二维数组 |

## 二、寄存器间接寻址规则

### 2.1 只有 4 个寄存器可以用 `[]` 间接寻址

**BX、SI、DI、BP**（记住口诀：**"补习弟 BP"** → BX/SI/DI/BP）

| 寄存器 | 默认段寄存器 | 典型场景 |
|--------|------------|---------|
| BX | DS | 数据区的基地址 |
| SI | DS | 源数据地址（串操作） |
| DI | ES | 目的数据地址（串操作） |
| BP | SS | 访问栈中参数 |

### 2.2 寄存器组合规则

**允许的组合**：
- `[BX + SI]` ✓
- `[BX + DI]` ✓
- `[BP + SI]` ✓
- `[BP + DI]` ✓

**禁止的组合**：
- `[SI + DI]` ✗（两个都是变址，没有基址）
- `[BX + BP]` ✗（两个都是基址，没有变址）

**规则**：必须是**一个基址寄存器（BX/BP）+ 一个变址寄存器（SI/DI）**。

### 2.3 完整格式

```
[BX/BP + SI/DI + 位移量]
```

位移量可以是 8 位或 16 位立即数（可省略）。

```asm
mov ax, [bx]           ; 最简单的寄存器间接寻址
mov ax, [bx + 4]       ; 基址寻址（访问结构体成员）
mov ax, [bx + si]      ; 基址变址寻址
mov ax, [bx + si + 6]  ; 基址变址 + 位移（最完整形式）
```

## 三、loop 指令与循环模式

### 3.1 loop 工作机制

```
(1) CX = CX - 1
(2) 如果 CX ≠ 0，跳转到标号处
    如果 CX = 0，执行下一条指令
```

### 3.2 标准循环模板

```asm
mov cx, 循环次数     ; CX 是 loop 的计数器
s:
    ; 循环体
    loop s           ; CX--; 若 CX≠0 则跳转到 s
```

**注意**：`loop` 是**段内短转移**（-128 ~ +127 字节范围内）。

### 3.3 完整示例：累加数组

```asm
; 计算内存 FFFF:0 ~ FFFF:B 共 12 个字节的和，存于 DX
mov ax, 0FFFFH
mov ds, ax
mov bx, 0          ; [bx] 作为偏移地址
mov cx, 12         ; 循环 12 次
mov dx, 0          ; 累加器清零
s:
    mov al, [bx]   ; 取出一个字节（8位）
    mov ah, 0       ; 高位置 0
    add dx, ax      ; 加到 16 位累加器
    inc bx          ; bx++，指向下一字节
    loop s
```

## 四、大小写转换经典题

利用 ASCII 码性质：大写字母第 5 位为 0，小写字母第 5 位为 1。

```asm
; 将字符串中的小写字母转为大写
; 'a'=61H (0110 0001) → 'A'=41H (0100 0001)
; 方法：AND AL, 11011111B (将第 5 位置 0)

mov cx, strlen
s:
    mov al, [si]           ; 取一个字符
    and al, 11011111B      ; 强制转为大写
    mov [di], al           ; 写回
    inc si
    inc di
    loop s
```

## 五、双重循环处理

loop 默认只用 CX 计数。双重循环需要**保存外层 CX**：

```asm
mov cx, 外层次数
outer:
    push cx              ; 保存外层 CX
    mov cx, 内层次数
    inner:
        ; 内层循环体
        loop inner
    pop cx               ; 恢复外层 CX
    loop outer
```

## 六、易错点

1. **不能用 `[ax]`、`[cx]`、`[dx]`**：只有 BX/BP/SI/DI 能用于间接寻址
2. **`[SI+DI]` 非法**：不能两个变址寄存器组合
3. **loop 触发条件**：CX 先减 1 再判断，所以 `mov cx, 0` 会导致循环 65536 次
4. **DS 的默认性**：`[bx]` 等价于 `DS:[bx]`，要访问其他段需显式指定
