# 标志寄存器与条件转移

> 一句话：标志寄存器是 CPU 的"状态报告单"。CMP + Jxx 是实现分支、循环判断的核心机制。

## 一、标志位速查

### 1.1 状态标志（ALU 自动设置）

| 位 | 名称 | 触发条件 | 口诀 |
|----|------|---------|------|
| 6 | ZF | 结果=0 → ZF=1 | "零则为 1" |
| 7 | SF | 结果最高位=1 → SF=1 | "负数则为 1" |
| 0 | CF | 无符号进位/借位 → CF=1 | "进位则为 1" |
| 11 | OF | 有符号溢出 → OF=1 | "溢出则为 1" |
| 2 | PF | 低8位有偶数个1 → PF=1 | "偶则为 1" |

### 1.2 控制标志（程序员设置）

| 位 | 名称 | 设置指令 | 清除指令 |
|----|------|---------|---------|
| 10 | DF | STD（DI/SI 递减） | CLD（DI/SI 递增） |
| 9 | IF | STI（允许中断） | CLI（禁止中断） |
| 8 | TF | 单步调试用 | — |

## 二、CMP 指令

```asm
cmp ax, bx          ; 计算 AX-BX，不保存结果，只影响标志寄存器
```

**执行后各标志位含义**：

| AX vs BX (无符号) | CF | ZF | AX vs BX (有符号) |
|-------------------|---|-----|---|
| AX < BX | 1 | 0 | — |
| AX = BX | 0 | 1 | — |
| AX > BX | 0 | 0 | — |

## 三、条件转移指令对照表

### 3.1 无符号数比较（Above/Below）

| 指令 | 条件 | 含义 |
|------|------|------|
| `je` / `jz` | ZF=1 | 等于 / 为零 |
| `jne` / `jnz` | ZF=0 | 不等于 / 非零 |
| `ja` / `jnbe` | CF=0 且 ZF=0 | 大于（Above） |
| `jae` / `jnb` | CF=0 | 大于等于 |
| `jb` / `jnae` | CF=1 | 小于（Below） |
| `jbe` / `jna` | CF=1 或 ZF=1 | 小于等于 |

### 3.2 有符号数比较（Greater/Less）

| 指令 | 条件 | 含义 |
|------|------|------|
| `jg` / `jnle` | ZF=0 且 SF=OF | 大于 |
| `jge` / `jnl` | SF=OF | 大于等于 |
| `jl` / `jnge` | SF≠OF | 小于 |
| `jle` / `jng` | ZF=1 或 SF≠OF | 小于等于 |

### 3.3 单标志判断

| 指令 | 条件 | 含义 |
|------|------|------|
| `jc` | CF=1 | 有进位 |
| `jnc` | CF=0 | 无进位 |
| `jo` | OF=1 | 有溢出 |
| `jno` | OF=0 | 无溢出 |
| `js` | SF=1 | 为负 |
| `jns` | SF=0 | 为正 |
| `jcxz` | CX=0 | CX 为零 |

## 四、转移距离

| 类型 | 指令 | 范围 |
|------|------|------|
| 段内短转移 | `jmp short` / `jxx` | -128 ~ +127 字节 |
| 段内近转移 | `jmp near ptr` | -32768 ~ +32767 |
| 段间远转移 | `jmp far ptr` | 任意 |

**jxx 条件跳转 = 段内短转移**！只能在标号前后 128 字节范围内跳转。

## 五、JMP 指令的所有形式

```
jmp short 标号           ; IP += 8位位移量
jmp near ptr 标号        ; IP += 16位位移量
jmp far ptr 标号         ; CS:IP = 标号所在段:偏移
jmp ax                  ; IP = AX
jmp word ptr [bx]       ; IP = DS:[BX]（内存间接跳转）
jmp dword ptr [bx]      ; CS:IP = DS:[BX]（段间间接跳转）
```

## 六、典型程序模式

### 6.1 if-else

```asm
cmp ax, 0
je  zero_case        ; if (ax==0)
; else 分支代码
jmp done
zero_case:
; then 分支代码
done:
```

### 6.2 while 循环

```asm
loop_start:
    cmp ax, 100
    jae loop_done      ; while (ax < 100)
    inc ax             ; 循环体
    jmp loop_start
loop_done:
```

### 6.3 for 循环（用 loop）

```asm
mov cx, 10
for_loop:
    ; 循环体（执行 10 次）
    loop for_loop
```

## 七、CF vs OF 深入理解

```
例1（无符号溢出）：AL=255, BL=1
    add al, bl   → AL=0, CF=1, OF=0
    对无符号数：255+1 应该得 256，但 AL 只有 8 位 → CF=1（进位）

例2（有符号溢出）：AL=127, BL=1
    add al, bl   → AL=128(80H), CF=0, OF=1
    对有符号数：127+1 应该得 128，但 8 位有符号范围 -128~127 → OF=1（溢出）

例3（两者都溢出）：AL=128, BL=128
    add al, bl   → AL=0, CF=1, OF=1
    80H+80H=100H → AL=00H, 无符号进位 + 有符号溢出
```
