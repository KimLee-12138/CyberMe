# 例题集：汇编核心

> 覆盖考试高频题型：物理地址计算、指令执行分析、程序阅读与编写。

---

## 题 1. 物理地址计算 ★★☆

**题目**：段地址=2000H，偏移地址=1234H。求物理地址。

**解**：2000H × 10H + 1234H = 20000H + 1234H = **21234H**。

---

## 题 2. 多对一映射 ★★☆

**题目**：写出物理地址 10000H 的两种不同段:偏移表示。

**解**：
- 1000H:0000H → 10000H + 0000H = 10000H
- 0FFFH:0010H → 0FFF0H + 0010H = 10000H

同一个物理地址可有无穷多种段:偏移表示。

---

## 题 3. 栈操作追踪 ★★☆

**题目**：初始 SP=0010H，依次执行 `push ax`、`push bx`、`pop cx`、`push dx`。求最终 SP 和栈顶内容。

**解**：
| 操作 | SP 变化 | 栈顶 |
|------|---------|------|
| 初始 | 0010H | — |
| push ax | 000EH | AX |
| push bx | 000CH | BX |
| pop cx | 000EH | AX（CX=BX） |
| push dx | 000CH | DX |

最终 SP = 000CH，栈顶为 DX。

## 题 4. 标志位分析 ★★★

**题目**：AL=80H, BL=80H。执行 `add al, bl` 后，AL=?，CF=?，OF=?，ZF=?，SF=?

**解**：
- 80H + 80H = 100H → AL = 00H（8位截断）
- CF = 1（无符号溢出：128+128=256 > 255）
- OF = 1（有符号溢出：-128 + (-128) = -256 < -128）
- ZF = 1（结果为 0）
- SF = 0（最高位为 0）

## 题 5. CMP + Jxx ★★★

**题目**：AX=5, BX=10。分析下列指令的跳转结果。

```asm
cmp ax, bx
jl  label1    ; 有符号：5<10 → 跳转 ✓
jb  label2    ; 无符号：5<10 → 跳转 ✓
jg  label3    ; 5>10 → 不跳转
```

---

## 题 6. 双重循环 ★★★

**题目**：外层循环 3 次、内层 5 次。写出汇编代码。

```asm
mov cx, 3
outer:
    push cx
    mov cx, 5
    inner:
        ; 执行 3×5=15 次
        loop inner
    pop cx
    loop outer
```

## 题 7. 大小写转换程序 ★★★

**题目**：编写程序将字符串 "HeLLo WoRLd!" 中的小写转大写、大写转小写。

**解**：
```asm
mov cx, len
lea bx, str
convert:
    mov al, [bx]
    cmp al, 'A'
    jb  next
    cmp al, 'Z'
    ja  check_lower
    or  al, 00100000B     ; 大写→小写（第5位置1）
    jmp store
check_lower:
    cmp al, 'a'
    jb  next
    cmp al, 'z'
    ja  next
    and al, 11011111B     ; 小写→大写（第5位置0）
store:
    mov [bx], al
next:
    inc bx
    loop convert
```

## 题 9. 分析 CS:IP 变化 ★★★

**题目**：执行以下代码，写出每一步 CS:IP 的值。

```asm
1000:0000  mov ax, 0
1000:0003  jmp 1000:0008
1000:0008  mov bx, ax
```

**解**：`jmp 1000:0008` 直接将 CS=1000H, IP=0008H（覆盖，不是位移）。跳转过程中不会执行 0003H 后面的指令。

---

## 题 10. loop 循环边界 ★★☆

**题目**：CX=0 时执行 `loop label` 会发生什么？

**解**：CX 先减 1 → CX=FFFFH → CX≠0 → 跳转到 label！循环体将执行 65536 次。

**教训**：不要用 `mov cx, 0` 作为 loop 的初始值。最小循环次数用 `jcxz` 提前判断。

---

## 题 11. 复杂寻址计算 ★★★

**题目**：DS=2000H, BX=100H, SI=50H。求 `mov ax, [bx+si+20H]` 访问的物理地址。

**解**：DS×10H + BX + SI + 20H = 20000H + 100H + 50H + 20H = **20170H**。

---

## 题 12. 除法溢出处理 ★★★

**题目**：AX=100H, BL=2。执行 `div bl` 后结果是什么？

**解**：
- AH=0, AL=80H（100H÷2=80H）
- 但如果 AX=FFFFH, BL=1 → 商=FFFFH > FFH(AL 范围) → **0 号中断（除法溢出）**
