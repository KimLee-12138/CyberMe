# 8086 汇编指令速查卡

> 考前快速浏览。按功能分类，标注影响标志位。

## 一、数据传送

| 指令 | 格式 | 说明 | 标志位 |
|------|------|------|--------|
| MOV | `mov dest, src` | src→dest | 不影响 |
| XCHG | `xchg op1, op2` | 交换 | 不影响 |
| LEA | `lea reg, mem` | 取偏移地址→reg | 不影响 |
| PUSH | `push src` | SP-=2, [SP]=src | 不影响 |
| POP | `pop dest` | dest=[SP], SP+=2 | 不影响 |

## 二、算术运算

| 指令 | 格式 | 说明 | OF|SF|ZF|CF|
|------|------|------|------|------|------|------|
| ADD | `add dest, src` | dest+=src | ✓|✓|✓|✓|
| ADC | `adc dest, src` | dest+=src+CF | ✓|✓|✓|✓|
| SUB | `sub dest, src` | dest-=src | ✓|✓|✓|✓|
| SBB | `sbb dest, src` | dest-=src-CF | ✓|✓|✓|✓|
| INC | `inc dest` | dest++ | ✓|✓|✓|✗|
| DEC | `dec dest` | dest-- | ✓|✓|✓|✗|
| MUL | `mul src` | 无符号乘 | ✓|—|—|✓|
| DIV | `div src` | 无符号除 | 未定义 |
| CMP | `cmp op1, op2` | op1-op2(不存) | ✓|✓|✓|✓|
| NEG | `neg dest` | 取负 | ✓|✓|✓|✓|

## 三、逻辑与移位

| 指令 | 格式 | 说明 |
|------|------|------|
| AND | `and dest, src` | 按位与 |
| OR | `or dest, src` | 按位或 |
| XOR | `xor dest, src` | 按位异或（`xor ax,ax`=清零） |
| NOT | `not dest` | 按位取反 |
| SHL | `shl dest, 1` | 逻辑左移（×2） |
| SHR | `shr dest, 1` | 逻辑右移 |
| SAR | `sar dest, 1` | 算术右移（保持符号） |

## 四、转移控制

| 指令 | 条件 | 说明 |
|------|------|------|
| JMP | 无 | 无条件跳转 |
| JE/JZ | ZF=1 | 等于/为零 |
| JNE/JNZ | ZF=0 | 不等于 |
| JA/JNBE | CF=0 & ZF=0 | 无符号 > |
| JB/JNAE | CF=1 | 无符号 < |
| JG/JNLE | ZF=0 & SF=OF | 有符号 > |
| JL/JNGE | SF!=OF | 有符号 < |
| JC | CF=1 | 有进位 |
| JNC | CF=0 | 无进位 |
| JO | OF=1 | 有溢出 |
| JNO | OF=0 | 无溢出 |
| JCXZ | CX=0 | CX为零 |
| LOOP | CX!=0 | CX--, 若 CX≠0 跳转 |
| LOOPE | CX!=0 & ZF=1 | CX--, 相等时循环 |
| CALL | — | 调用子程序 |
| RET | — | 返回 |

## 五、中断与系统

| 指令 | 说明 |
|------|------|
| INT n | 调用 n 号中断 |
| IRET | 中断返回 |
| CLI | 关中断 (IF=0) |
| STI | 开中断 (IF=1) |
| CLD | DF=0 (正向) |
| STD | DF=1 (反向) |

## 六、常用 INT 21H 功能

| AH | 功能 | 入口 |
|----|------|------|
| 01H | 键盘输入(回显) | 返回 AL=ASCII |
| 02H | 显示字符 | DL=字符 |
| 09H | 显示字符串 | DS:DX=字符串(以$结尾) |
| 0AH | 字符串输入 | DS:DX=缓冲区 |
| 4CH | 程序退出 | AL=返回码 |

## 七、乘法表

| 操作数 | 隐含被乘数 | 结果 |
|--------|-----------|------|
| 8位(mul bl) | AL | AX |
| 16位(mul bx) | AX | DX:AX |

## 八、除法表

| 操作数 | 隐含被除数 | 商 | 余数 |
|--------|-----------|-----|------|
| 8位(div bl) | AX | AL | AH |
| 16位(div bx) | DX:AX | AX | DX |
