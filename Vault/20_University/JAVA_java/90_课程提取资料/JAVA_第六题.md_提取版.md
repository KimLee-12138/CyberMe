# 第六题

不能构成重载

 重载要求方法名相同，但是参数列表必须不同，方法一和方法二的参数列表都是double，所以无法构成函数重载

重载：发生在同一个类中

方法覆盖：发生在父类和子类之间

重载：方法名相同，参数列表必须不同

方法重载：方法名相同，参数列表必须相同

重载：返回类型可以修改

方法重载：返回类型必须相同
