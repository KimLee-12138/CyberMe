# 第四题

final:

- 修饰数据成员：数据成员变为常量，其值不可以被修改
- 修饰方法：该方法仍可以被子类继承，但是不能被重载
- 修饰类：该类不可以被继承

finally:

- finally配合try-catch使用，表示无论是否捕获异常，finally中的代码块都会被执行，一般用于资源的释放、IO流的关闭，保证资源不会泄露
