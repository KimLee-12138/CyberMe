# 通用学术英语 EAP 

---

## 1. 论文结构必背：IMRD + Abstract

| 部分 | 主要功能 | 高频考法 |
|---|---|---|
| **Introduction** | 介绍背景、前人研究、研究空白、研究目的/问题 | 不能详细分析结果 |
| **Methods** | 写研究对象/材料、实验或调查过程、数据收集、统计分析工具 | 要 clear, concise, logical；别人能 replicate |
| **Results** | 客观呈现主要发现、数据、图表 | 只报告结果，不重点解释原因 |
| **Discussion** | 解释结果、对比前人研究、说明意义、指出局限和未来研究 | may be due to / implications / limitations |
| **Conclusion** | 总结贡献、应用意义、局限、建议 | so what? what next? |
| **Abstract** | 背景 + 目的 + 方法 + 结果 + 结论/意义 | 样卷要求 180–200 words，不能照抄原文 |

**Abstract 五件套：** Background → Purpose → Methods → Results → Conclusion / Implications 
**摘要写作核心：** 保留关键数据，压缩原文，换表达，不写 I think。

## 3. Part IV Synthesis Paragraph：三篇资料合成一段

**它不是三篇分别总结，而是找三篇资料之间的关系。** 关系一般有：相同、不同、补充、原因、结果、解决办法。

### 写作步骤
1. 先给三篇资料贴标签：Source 1 = 优点/原因；Source 2 = 缺点/问题；Source 3 = 方法/解决。  
2. 第一句写总观点：The three sources suggest that...  
3. 中间整合三篇资料：Smith says A, while Lee says B, Wang further shows C.  
4. 最后一句下综合结论：Taken together / Therefore...

### 万能模板（约120词）
Overall, the three sources suggest that **[topic]** should be understood as **[overall insight]** rather than a single issue. **Smith (2024)** emphasizes that **[Source 1 main idea]**, showing the importance of **[key point]**. However / Similarly, **Lee (2023)** points out that **[Source 2 main idea]**, which indicates **[relationship: problem / support / contrast]**. **Wang (2022)** further extends this view by arguing that **[Source 3 main idea]**. Taken together, these sources indicate that **[综合结论]**. Therefore, a more balanced / comprehensive approach is needed to **[solution or implication]**.

**常用 reporting verbs：** suggests, argues, reports, shows, finds, highlights, points out, demonstrates, warns.

## 4. Part V Abstract Writing：180–200词结构式摘要

### 必背框架
1. **Background：** 研究背景和问题  
2. **Purpose：** 本研究想解决什么  
3. **Methods：** 研究对象、数据来源、时间地点、分析方法  
4. **Results：** 主要发现，保留关键数字  
5. **Conclusion：** 意义、应用、局限/未来研究

### 万能模板
**[Background]** X has become an important issue because it causes **[problem]**. **[Purpose]** This study aimed to examine / evaluate / quantify whether **[method/intervention]** could **[effect]**. **[Methods]** Data were collected from **[subjects/materials/sites]** during **[time]**. The researchers measured **[variables]** and analyzed the data using **[method/tool]**. **[Results]** The results showed that **[major finding + data]**. In addition, **[second finding]**. **[Conclusion]** These findings suggest that **[meaning/application]**. However, because the study was limited by **[limitation]**, future research should **[future direction]**.

### 摘要高分提醒
- 不要逐句抄原文，要 paraphrase。  
- Results 里一定写具体发现，给数字就保留数字。  
- Discussion/Conclusion 要写意义：These findings suggest/support/indicate...  
- 不写个人评价：I think / in my opinion 全删。  
- 控制 180–200 words，推荐 185 左右。

---

## 5. Part I 判断题 + 学术写作高频点

**正式学术风格：**  
- 避免：I think, we/you 过多, OK, lots of, rhetorical questions, contractions, proverbs。  
- 使用：This study / The results / The findings / numerous / significant / indicate / demonstrate。  
- 被动语态常用来强调 action/result，不强调 doer。  
- **Nominalization 名词化：** turn verbs/adjectives/clauses into nouns/nominal phrases，使表达更正式、更紧凑。例：analyze → analysis；decide → decision；because... → due to...
- **Impersonal subject 非人称主语：** It is found that... / The results show... / This study examines...，比 We found / You can see 更正式。
- **Cohesion & coherence：** 句子之间要有连接；同一段主题要一致；用 however, therefore, moreover, in contrast, taken together。



# EAP 论文五部分要素

---

## 一、Introduction 引言：回答“为什么要做这个研究？”

### 1. 主要要素
1. **Specific topic / background**：引出具体研究主题，说明研究背景。  
2. **Previous studies / literature review**：介绍前人研究已经做了什么。  
3. **Research gap**：指出还有什么没有解决、没有研究清楚。  
4. **Research aim / purpose**：说明本研究要做什么。  
5. **Research questions**：提出研究问题，说明研究要回答什么。

### 2. 常见信号词
- **背景**：In recent years..., X has attracted increasing attention...
- **前人研究**：Previous studies have shown / found / suggested...
- **研究空白**：However, little is known about..., remains unclear / underexplored...
- **研究目的**：This study aims to..., The purpose of this study is to...
- **研究问题**：This study addresses the following questions...

### 3. 考试判断
看到 **In recent years / has attracted attention** 多半是背景。  
看到 **Previous studies / Smith et al.** 多半是前人研究。  
看到 **However / little is known / remains underexplored** 多半是研究空白。  
看到 **aim / purpose / objective / examine / investigate** 多半是研究目的。

---

## 二、Methods 方法：回答“研究是怎么做的？”

### 1. 主要要素
1. **Research subjects / participants**：研究对象、参与者、样本。  
2. **Research design**：研究设计、实验流程、调查过程。  
3. **Data collection**：数据如何收集，包括抽样方式、选择标准、设备、软件。  
4. **Data analysis**：数据如何分析，包括统计方法、数学模型、理论框架、分析软件。  
5. **Ethical approval**：伦理审批，尤其是涉及人的实验或调查。  
6. **Replicability**：方法要完整、简洁、逻辑清楚，使别人能够重复研究。

### 2. 常见信号词
- **参与者**：Participants included..., A total of ... students / patients were recruited...
- **数据收集**：Data were collected from..., Measurements were recorded...
- **方法/设计**：This study used / employed / adopted...
- **统计分析**：Data were analyzed using SPSS / t-tests / regression analysis...
- **伦理审批**：Ethical approval was obtained from...

### 3. 考试判断
看到 **participants / subjects / samples** → 参与者。  
看到 **collected / measured / recorded / conducted** → 方法或数据收集。  
看到 **SPSS / t-test / ANOVA / regression** → 统计分析工具。  
看到 **ethical approval / informed consent** → 伦理审批。

---

## 三、Results 结果：回答“研究发现了什么？”

### 1. 主要要素
1. **Major findings**：客观呈现主要研究发现。  
2. **Answer to aims / hypotheses / questions**：结果要回应研究目的、假设或研究问题。  
3. **Representative results**：呈现有代表性的结果，不是堆所有数据。  
4. **Logical sequence**：按照逻辑顺序组织结果。  
5. **Figures / tables / diagrams / written texts**：结果常通过图、表、图示和文字说明呈现。

### 2. 常见信号词
- The results showed that...
- The findings indicated that...
- It was found that...
- Results revealed significant differences / gains...
- The data suggest that...

### 3. 考试判断
Results 部分只强调 **发现什么**，一般不深入解释原因。  
看到 **showed / found / indicated / revealed / significant** 多半是总结主要发现。

---
