# 党员义务与权利

> 一句话：党员既是先锋战士又是普通劳动者，享八项权利，尽八项义务。

## 一、入党条件

- 年满**十八岁**
- 承认党的**纲领和章程**
- 愿意参加**党的一个组织**并在其中积极工作
- 执行党的**决议**
- 按期**交纳党费**

## 二、入党誓词

"我志愿加入中国共产党，拥护党的纲领，遵守党的章程，履行党员义务，执行党的决定，严守党的纪律，保守党的秘密，对党忠诚，积极工作，为共产主义奋斗终身，随时准备为党和人民牺牲一切，永不叛党。"

## 三、党员八项义务

1. 认真学习**理论**和科学文化知识
2. 贯彻执行党的**基本路线和方针政策**
3. 坚持**党和人民的利益高于一切**
4. 自觉遵守**党的纪律**，模范遵守国家法律法规
5. 维护**党的团结和统一**
6. 切实开展**批评与自我批评**
7. 密切联系**群众**
8. 发扬社会主义**新风尚**，带头实践社会主义核心价值观

**记忆口诀**：学习-路线-利益-纪律-团结-批评-群众-新风尚

## 四、党员八项权利

1. 参加党的会议，阅读党的文件，接受党的**教育和培训**
2. 参加**政策问题讨论**
3. 对党的工作提出**建议和倡议**
4. 有根据地对党组织和党员进行**批评、揭发、检举**
5. 行使**表决权、选举权**，有被选举权
6. 在党组织讨论党纪处分或鉴定时，本人有权**参加和申辩**
7. 对党的决议和政策有不同意见，可**保留**并向上级提出
8. 向上级提出**请求、申诉、控告**

## 五、预备党员

- 预备期：**一年**
- 预备党员的义务同正式党员
- 预备党员没有**表决权、选举权和被选举权**
- 预备期满：支部大会讨论转正

## 六、党的纪律处分

| 种类 | 说明 |
|------|------|
| **警告** | 最轻微的处分 |
| **严重警告** | 比警告重一级 |
| **撤销党内职务** | 撤销担任的党内职务 |
| **留党察看** | 最长不超过两年 |
| **开除党籍** | 党内的最高处分 |
