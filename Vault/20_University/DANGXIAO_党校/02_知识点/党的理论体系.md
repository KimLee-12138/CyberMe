# 党的理论体系

> 一句话：马克思主义中国化产生了两大理论成果——毛泽东思想和中国特色社会主义理论体系。

## 一、马克思列宁主义

**马克思主义的三个组成部分**：
- **哲学**（辩证唯物主义和历史唯物主义）
- **政治经济学**（剩余价值学说）
- **科学社会主义**

**列宁主义**：帝国主义是无产阶级革命的前夜，社会主义可在一国首先胜利。

## 二、毛泽东思想

### 2.1 活的灵魂

| 方面 | 含义 |
|------|------|
| **实事求是** | 一切从实际出发，理论联系实际 |
| **群众路线** | 一切为了群众，一切依靠群众 |
| **独立自主** | 走自己的路 |

### 2.2 三大法宝

**统一战线、武装斗争、党的建设**。

- 统一战线：团结一切可以团结的力量
- 武装斗争：枪杆子里出政权（八七会议提出）
- 党的建设：思想建党、政治建军（古田会议确立）

### 2.3 历史地位

七大确立为党的指导思想。回答了"什么是新民主主义革命、怎样进行新民主主义革命"。

## 三、邓小平理论

### 3.1 核心命题

**"什么是社会主义、怎样建设社会主义"**

### 3.2 关键论断

| 论断 | 内容 |
|------|------|
| 社会主义本质 | 解放生产力，发展生产力，消灭剥削，消除两极分化，最终达到共同富裕 |
| 发展才是硬道理 | 南方谈话(1992) |
| 三个有利于 | 是否有利于发展生产力、增强综合国力、提高人民生活水平 |
| 一国两制 | 香港、澳门回归的成功实践 |
| 社会主义初级阶段 | 十三大提出，至少一百年 |

### 3.3 历史地位

十五大确立为党的指导思想，写入党章。

## 四、"三个代表"重要思想

| 代表 | 内涵 |
|------|------|
| 先进生产力 | 发展是硬道理 |
| 先进文化 | 精神文明建设 |
| 最广大人民根本利益 | 执政为民 |

**地位**：立党之本、执政之基、力量之源。

回答"建设什么样的党、怎样建设党"。

## 五、科学发展观

- **第一要义**：发展
- **核心**：以人为本
- **基本要求**：全面协调可持续
- **根本方法**：统筹兼顾

回答"实现什么样的发展、怎样发展"。

## 六、习近平新时代中国特色社会主义思想

### 6.1 核心要义

坚持和发展中国特色社会主义。

### 6.2 "八个明确"（核心）

1. 坚持和发展中国特色社会主义的总任务
2. 新时代我国社会主要矛盾：人民日益增长的美好生活需要 vs 不平衡不充分的发展
3. 总体布局(五位一体)和战略布局(四个全面)
4. 全面深化改革总目标
5. 全面推进依法治国总目标
6. 党在新时代的强军目标
7. 中国特色大国外交：构建**人类命运共同体**
8. 中国特色社会主义最本质的特征：**党的领导**

### 6.3 历史地位

十九大确立为党的指导思想，写入党章。实现了马克思主义中国化的**第三次历史性飞跃**。

## 七、理论演进脉络

```
马列主义 → 毛泽东思想(七大1945)
    → 邓小平理论(十五大1997)
    → "三个代表"(十六大2002)

```
