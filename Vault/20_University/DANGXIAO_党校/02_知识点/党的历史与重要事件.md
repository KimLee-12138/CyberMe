# 党的历史与重要事件

> 中国共产党成立于 1921 年，历经新民主主义革命、社会主义革命和建设、改革开放和社会主义现代化建设、中国特色社会主义新时代四个历史时期。

## 一、新民主主义革命时期（1921-1949）

### 1.1 党的创立与早期革命

| 时间 | 事件 | 意义 |
|------|------|------|
| 1921.7 | **中共一大**（上海→嘉兴南湖） | 党的诞生 |
| 1922.7 | 中共二大 | 制定民主革命纲领 |
| 1923.6 | 中共三大 | 决定与国民党合作 |
| 1924-1927 | **第一次国共合作**（党内合作） | 北伐战争 |
| 1927.8.1 | **南昌起义** | 武装反抗国民党第一枪 |
| 1927.8.7 | **八七会议** | "枪杆子里出政权" |
| 1927.9 | **秋收起义**→三湾改编→井冈山 | 农村包围城市道路的开端 |
| 1929.12 | **古田会议** | 思想建党、政治建军 |
| 1931.11 | **中华苏维埃共和国**成立 | 瑞金，临时中央政府 |
| 1934.10-1936.10 | **红军长征** | 战略大转移，遵义会议（1935.1）确立毛泽东领导地位 |
| 1936.12 | **西安事变**和平解决 | 抗日民族统一战线初步形成 |

### 1.2 抗日战争时期（1937-1945）

- **第二次国共合作**（1937-1945）：以国共两个政权、两个军队合作为基础，**党外合作**，**没有一个统一协调的正式机构**，**没有一个共同承认的政治纲领**
- 1937.9：抗日民族统一战线正式形成
- 1940.8-12：**百团大战**
- 1945.4-6：**中共七大**——确立**毛泽东思想**为党的指导思想
- 1945.9：抗日战争胜利

### 1.3 解放战争时期（1945-1949）

- 1945.10：重庆谈判
- 1948.9-1949.1：辽沈、淮海、平津**三大战役**
- 1949.3：**七届二中全会**——"两个务必"
- 1949.10.1：**中华人民共和国成立**

## 二、社会主义革命和建设时期（1949-1978）

| 时间 | 事件 |
|------|------|
| 1950-1953 | 抗美援朝 |
| 1953-1957 | **"一五"计划**（社会主义工业化起步） |
| 1956 | **中共八大**（社会主义改造基本完成） |
| 1958 | "大跃进"和**人民公社化运动**（"左"倾错误严重泛滥） |
| 1966-1976 | **"文化大革命"**（十年内乱） |
| 1971.10 | 恢复联合国合法席位 |
| 1976.10 | 粉碎"四人帮"结束文革 |

## 四、中国特色社会主义新时代（2012-至今）

| 时间 | 事件 |
|------|------|
| 2012.11 | 中共十八大——**习近平当选总书记** |
| 2017.10 | **中共十九大**——**习近平新时代中国特色社会主义思想**写入党章 |
| 2020 | 全面建成小康社会 |
| 2021.7 | 建党百年 |
| 2022.10 | **中共二十大**——中国式现代化 |

## 五、重要考点速记

1. **两个先锋队**：中国工人阶级的先锋队 + 中国人民和中华民族的先锋队
2. **国共两次合作**：第一次党内合作(1924-1927)，第二次党外合作(1937-1945)
3. **三大战役**：辽沈、淮海、平津
4. **三次历史性飞跃**：毛泽东思想→中国特色社会主义理论体系→习近平新时代中国特色社会主义思想
5. **四项基本原则**：坚持社会主义道路、人民民主专政、党的领导、马列主义毛泽东思想（**核心：党的领导**）
