# 党的基本知识

> 一句话：党章是党的总章程。掌握党的性质、宗旨、指导思想、组织原则和党员权利义务。

## 一、党的性质

**"两个先锋队"**：
- 中国工人阶级的先锋队
- 中国人民和中华民族的先锋队

**"一个领导核心"**：中国特色社会主义事业的领导核心

**"三个代表"**：代表中国先进生产力的发展要求、代表中国先进文化的前进方向、代表中国最广大人民的根本利益

## 二、党的宗旨

**全心全意为人民服务**（党的最高价值取向，一切工作的出发点和归宿）

毛泽东："我们共产党人好比种子，人民好比土地。"

## 三、党的指导思想

| 思想/理论 | 确立时间 | 核心内容 |
|----------|---------|---------|
| **马克思列宁主义** | 建党之初 | 科学社会主义理论基础 |
| **毛泽东思想** | 七大(1945) | 实事求是、群众路线、独立自主 |
| **邓小平理论** | 十五大(1997) | 改革开放、社会主义本质论、初级阶段 |
| **"三个代表"重要思想** | 十六大(2002) | 立党之本、执政之基、力量之源 |
| **科学发展观** | 十七大(2007)/十八大(2012) | 第一要义是发展，核心是以人为本，基本要求是全面协调可持续 |
| **习近平新时代中国特色社会主义思想** | 十九大(2017) | 新时代坚持和发展中国特色社会主义 |

## 四、党的基本路线

**"一个中心，两个基本点"**：
- 以**经济建设**为中心
- 坚持**四项基本原则**（立国之本）
- 坚持**改革开放**（强国之路）

## 五、四项基本原则

| 序号 | 内容 |
|------|------|
| 1 | 坚持社会主义道路 |
| 2 | 坚持人民民主专政 |
| 3 | **坚持中国共产党的领导（核心）** |
| 4 | 坚持马克思列宁主义毛泽东思想 |

## 六、党的组织原则

**民主集中制**：民主基础上的集中 + 集中指导下的民主。

**四个服从**：
- 个人服从组织
- 少数服从多数
- 下级服从上级
- 全党服从中央

## 七、党的三大作风

| 作风 | 说明 |
|------|------|
| 理论联系实际 | 不搞教条主义 |
| 密切联系群众 | 从群众中来，到群众中去 |
| 批评与自我批评 | 党的自我革命 |

## 八、党的纪律

- **政治纪律**：最根本、最重要的纪律
- **组织纪律**：四个服从
- **廉洁纪律**：反腐倡廉
- **群众纪律**：维护群众利益
- **工作纪律**：尽职尽责
- **生活纪律**：品德端正

## 九、党的群众路线

一切为了群众，一切依靠群众，从群众中来，到群众中去。

群众路线是马克思主义认识论和历史唯物主义基本原则的体现。

## 十、重要概念速记

| 概念 | 答案 |
|------|------|
| 党的最高理想和最终目标 | 实现共产主义 |
| 党的根本宗旨 | 全心全意为人民服务 |
| "三个代表"的地位 | 立党之本、执政之基、力量之源 |
| 四项基本原则的核心 | 坚持党的领导 |
| 判断改革的标准 | "三个有利于" |
| 科学发展观的基本要求 | 全面协调可持续 |
