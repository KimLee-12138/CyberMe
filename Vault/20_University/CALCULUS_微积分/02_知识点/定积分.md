# 定积分

> 一句话：定积分是黎曼和的极限，计算面积、体积、弧长的核心工具。

## 一、定积分的定义

$$\int_a^b f(x)dx = \lim_{\lambda \to 0} \sum_{i=1}^n f(\xi_i)\Delta x_i$$

其中 $\lambda = \max_i \Delta x_i$（分割的最大子区间长度）。

**几何意义**：$f(x) \ge 0$ 时为曲边梯形面积；$f(x)$ 变号时为面积的代数和。

## 二、定积分的性质

1. $\int_a^a f(x)dx = 0$
2. $\int_a^b f(x)dx = -\int_b^a f(x)dx$
3. **区间可加性**：$\int_a^c f + \int_c^b f = \int_a^b f$
4. **线性性**：$\int_a^b (\alpha f + \beta g) = \alpha\int_a^b f + \beta\int_a^b g$
5. **单调性**：$f \le g$ 则 $\int f \le \int g$
6. **估值定理**：$m(b-a) \le \int_a^b f(x)dx \le M(b-a)$（$m, M$ 为最值）
7. **积分中值定理**：$\exists \xi \in [a,b]: \int_a^b f = f(\xi)(b-a)$

### 2.1 对称区间上的积分

- $f(x)$ 是偶函数：$\int_{-a}^a f = 2\int_0^a f$
- $f(x)$ 是奇函数：$\int_{-a}^a f = 0$

## 三、变上限积分与微积分基本定理

### 3.1 变上限积分函数

$$\Phi(x) = \int_a^x f(t)dt$$

**微积分学第一基本定理**：
$$\Phi'(x) = f(x)$$
即变上限积分的导数 = 被积函数在上限处的值。

### 3.2 推广（莱布尼兹公式）

$$\frac{d}{dx}\int_{a(x)}^{b(x)} f(t)dt = f(b(x))\cdot b'(x) - f(a(x))\cdot a'(x)$$

### 3.3 牛顿-莱布尼兹公式（N-L 公式）

$$\int_a^b f(x)dx = F(b) - F(a) = \Big[F(x)\Big]_a^b$$

其中 $F'(x) = f(x)$。**N-L 公式是微积分学的核心桥梁**。

## 四、定积分的换元法

令 $x = \varphi(t)$，则：
$$\int_a^b f(x)dx = \int_\alpha^\beta f(\varphi(t))\varphi'(t)dt$$

其中 $\varphi(\alpha) = a, \varphi(\beta) = b$。

**关键**：换元时必须**同时变换积分上下限**，且积分完成后**不需要换回原变量**。

例：$\int_0^{\pi/2} \sin^2 x \cos x dx$，令 $u = \sin x$，$du = \cos x dx$。$x=0 \to u=0$，$x=\pi/2 \to u=1$。原式 $= \int_0^1 u^2 du = \frac{1}{3}$。

## 五、定积分的分部积分法

$$\int_a^b u dv = \Big[uv\Big]_a^b - \int_a^b v du$$

例：$\int_0^{\pi} x\sin x dx = [-x\cos x]_0^\pi + \int_0^\pi \cos x dx = \pi + 0 = \pi$

## 六、反常积分

### 6.1 无穷区间反常积分

$$\int_a^{+\infty} f(x)dx = \lim_{b \to +\infty} \int_a^b f(x)dx$$

若极限存在则**收敛**，否则**发散**。

**重要结论（p-积分）**：
$$\int_1^{+\infty} \frac{1}{x^p}dx: \quad p>1 \text{ 收敛}, \quad p \le 1 \text{ 发散}$$

例：$\int_1^\infty \frac{1}{x^2}dx = 1$（收敛），$\int_1^\infty \frac{1}{x}dx = \infty$（发散）。

### 6.2 无界函数的反常积分（瑕积分）

若 $x=a$ 为瑕点（$\lim_{x \to a^+}f(x)=\infty$）：
$$\int_a^b f(x)dx = \lim_{\varepsilon \to 0^+} \int_{a+\varepsilon}^b f(x)dx$$

**比较判别法**：若 $0 \le f(x) \le g(x)$ 且 $\int g$ 收敛，则 $\int f$ 收敛。

## 七、定积分的几何应用

### 7.1 平面图形的面积

**直角坐标**：
$$A = \int_a^b [f_{\text{上}}(x) - f_{\text{下}}(x)]dx$$

**极坐标**：
$$A = \frac{1}{2}\int_\alpha^\beta r^2(\theta)d\theta$$

### 7.2 旋转体的体积

**绕 $x$ 轴（圆盘法）**：
$$V = \pi \int_a^b [f(x)]^2 dx$$

**绕 $y$ 轴（柱壳法）**：
$$V = 2\pi \int_a^b x f(x) dx \quad (f(x) \ge 0)$$

**绕 $y$ 轴（圆盘法重写）**：$x = g(y), y\in[c,d]$：
$$V = \pi \int_c^d [g(y)]^2 dy$$

### 7.3 曲线的弧长

**直角坐标**：$s = \int_a^b \sqrt{1 + [f'(x)]^2} dx$

**参数方程**：$s = \int_{t_1}^{t_2} \sqrt{[x'(t)]^2 + [y'(t)]^2} dt$

**极坐标**：$s = \int_\alpha^\beta \sqrt{[r(\theta)]^2 + [r'(\theta)]^2} d\theta$

### 7.4 旋转体的侧面积

绕 $x$ 轴：$S = 2\pi \int_a^b f(x)\sqrt{1+[f'(x)]^2}dx$

## 八、重要积分结论

| 结论 | 值 |
|------|-----|
| $\int_0^{\pi/2} \sin^n x dx = \int_0^{\pi/2} \cos^n x dx$ | $\frac{n-1}{n} \cdot \frac{n-3}{n-2} \cdots \frac{1}{2}\cdot\frac{\pi}{2}$ ($n$偶) 或 $\frac{n-1}{n} \cdots \frac{2}{3}\cdot 1$ ($n$奇) |
| $\int_0^\pi \sin^n x dx$ | $= 2\int_0^{\pi/2} \sin^n x dx$ |
| $\int_{-\infty}^\infty e^{-x^2}dx$ | $= \sqrt{\pi}$（高斯积分） |

## 九、定积分的近似计算

1. **矩形法**：$\int_a^b f \approx \frac{b-a}{n}\sum_{i=1}^n f(x_i^*)$
2. **梯形法**：$\int_a^b f \approx \frac{b-a}{2n}[f(x_0)+2\sum_{i=1}^{n-1}f(x_i)+f(x_n)]$
3. **辛普森法**：更高精度，抛物线逼近

## 十、易错点

1. **定积分换元必须同时换上下限**，不需要换回原变量
2. **对称区间上奇函数积分 = 0**：$x$、$\sin x$、$x^3$ 等
3. **瑕积分必须用极限处理**：$\int_{-1}^1 \frac{1}{x}dx$ 在 $x=0$ 处是瑕积分，不能直接用 N-L 公式
4. **变量代换时的单调性**：$x=\varphi(t)$ 必须单调以保证反函数存在
