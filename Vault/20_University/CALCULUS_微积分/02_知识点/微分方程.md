# 微分方程

> 一句话：含未知函数及其导数的方程。识别类型→套对应解法。

## 一、基本概念

**阶**：最高阶导数的阶数。**通解**含独立常数，个数=阶数。**特解**由初始条件确定。

## 二、一阶微分方程

### 2.1 可分离变量
$$\frac{dy}{dx} = f(x)g(y) \Rightarrow \int \frac{dy}{g(y)} = \int f(x)dx$$

### 2.2 齐次方程
$$\frac{dy}{dx} = \varphi(\frac{y}{x})$$

令 $u = y/x$，$y = ux, y' = u + xu'$，化为 $\frac{du}{\varphi(u)-u} = \frac{dx}{x}$

### 2.3 一阶线性
$$y' + P(x)y = Q(x)$$

**通解公式**：$y = e^{-\int Pdx}\left[\int Q e^{\int Pdx}dx + C\right]$

简记：$y = \frac{1}{\mu}(\int \mu Q dx + C)$，$\mu = e^{\int Pdx}$（积分因子）

例：$y' + 2xy = x$。$\mu = e^{x^2}$，$y = e^{-x^2}[\int xe^{x^2}dx + C] = \frac{1}{2}+Ce^{-x^2}$

### 2.4 伯努利方程
$$y' + Py = Q y^n \quad (n \neq 0,1)$$

令 $z = y^{1-n}$ → $z' + (1-n)Pz = (1-n)Q$ → 化为一阶线性。

## 三、可降阶的高阶方程

| 类型 | 特征 | 降阶法 |
|------|------|--------|
| $y^{(n)} = f(x)$ | 只含 $x$ | 逐次积分 $n$ 次 |
| $y'' = f(x, y')$ | **不显含 $y$** | 令 $p=y'$，$y''=p'$ |
| $y'' = f(y, y')$ | **不显含 $x$** | 令 $p=y'$，$y'' = p\frac{dp}{dy}$ |

**关键判断**：看方程中缺了什么。缺 $y$→令 $p=y'$；缺 $x$→令 $p=y'$ 且 $y''=p\frac{dp}{dy}$。

## 四、二阶常系数线性齐次方程

$$y'' + py' + qy = 0$$

**特征方程**：$r^2 + pr + q = 0$

| $\Delta$ | 根的类型 | 通解形式 |
|----------|---------|---------|
| $\Delta > 0$ | 两不等实根 $r_1, r_2$ | $y = C_1 e^{r_1x} + C_2 e^{r_2x}$ |
| $\Delta = 0$ | 相等实根 $r$ | $y = (C_1 + C_2 x)e^{rx}$ |
| $\Delta < 0$ | 共轭复根 $r = \alpha \pm i\beta$ | $y = e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)$ |

### 4.1 例题

1. $y'' - 5y' + 6y = 0$：$r^2-5r+6=0$，$r_1=2, r_2=3$，$y = C_1e^{2x}+C_2e^{3x}$

2. $y'' - 4y' + 4y = 0$：$r^2-4r+4=0$，$r=2$（二重），$y = (C_1+C_2x)e^{2x}$

3. $y'' + 4y' + 13y = 0$：$r = -2 \pm 3i$，$y = e^{-2x}(C_1\cos 3x + C_2\sin 3x)$

## 五、二阶常系数线性非齐次方程

$$y'' + py' + qy = f(x)$$

通解 = 齐次通解 $y_h$ + 非齐次特解 $y_p$。

### 5.1 待定系数法

| $f(x)$ 形式 | 特解 $y_p$ 形式 | 若与特征根重合 |
|------------|---------------|---------------|
| $P_n(x)$ ($n$ 次多项式) | $Q_n(x)$ ($n$ 次待定多项式) | 乘 $x^s$ |
| $Ae^{kx}$ | $C x^s e^{kx}$ | $s$ = $k$ 作为特征根的重数 |
| $A\cos\omega x + B\sin\omega x$ | $x^s(C_1\cos\omega x + C_2\sin\omega x)$ | $s$ = $\omega i$ 作为特征根的重数 |

**$s$ 的含义**：特征根中匹配的次数。$s=0$（不重合），$s=1$（单重根），$s=2$（二重根）。

### 5.2 例题

$y'' - 3y' + 2y = e^x$

齐次通解：$r^2-3r+2=0 \Rightarrow r_1=1, r_2=2 \Rightarrow y_h = C_1e^x+C_2e^{2x}$

$f(x)=e^x$，$k=1$ 是单重特征根 → 特解设为 $y_p = Axe^x$

代入：$y_p'=Ae^x(1+x), y_p''=Ae^x(2+x)$。代入原方程解得 $A=-1$。

通解：$y = C_1e^x + C_2e^{2x} - xe^x$

### 5.3 $f(x)$ 为两项之和

分别设特解再相加（叠加原理）。

例：$y'' + y = e^x + \sin x$ → $y_{p1}$ 针对 $e^x$，$y_{p2}$ 针对 $\sin x$，$y_p = y_{p1}+y_{p2}$

## 六、欧拉方程（选学）

$$x^2y'' + pxy' + qy = f(x)$$

令 $x = e^t$，化为常系数方程。对应 $x^k y^{(k)} = D(D-1)\cdots(D-k+1)y$，其中 $D = d/dt$。

## 七、解题流程图

```
识别方程类型
   ├── 一阶？
   │    ├── 可分离？ → 直接积分
   │    ├── 齐次？→ 令 u=y/x
   │    ├── 线性 y'+Py=Q？→ 积分因子法
   │    └── 伯努利？→ 令 z=y^{1-n}
   └── 二阶？
        ├── 不显含 y 或 x？→ 降阶
        ├── 常系数齐次？→ 特征方程
        └── 常系数非齐次？→ 待定系数法
```

## 八、易错点

1. **一阶线性公式中 $P(x)$ 前是 + 号**：$y'+Py=Q$，不是 $y'-Py$
2. **复根通解别忘了 $e^{\alpha x}$ 因子**
3. **待定系数法必须检查 $f(x)$ 的指数/频率是否与特征根重合**
4. **可分离变量时实数常数解不要遗漏**（如 $y=0$ 可能是特解）
5. **叠加原理只适用于线性非齐次方程**
