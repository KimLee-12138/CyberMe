# 不定积分

> 一句话：不定积分是求导的逆运算，找出所有导数等于被积函数的原函数。

## 一、定义与性质

若 $F'(x) = f(x)$，则 $\int f(x)dx = F(x) + C$。

- $(\int f(x)dx)' = f(x)$（积分再求导还原）
- $\int F'(x)dx = F(x) + C$（求导再积分差常数）
- $\int [\alpha f(x) + \beta g(x)]dx = \alpha\int f(x)dx + \beta\int g(x)dx$

## 二、基本积分公式全表

### 2.1 幂函数与指数对数

| 积分 | 结果 | 条件 |
|------|------|------|
| $\int x^n dx$ | $\frac{x^{n+1}}{n+1} + C$ | $n \neq -1$ |
| $\int \frac{1}{x}dx$ | $\ln|x| + C$ | $x \neq 0$ |
| $\int e^x dx$ | $e^x + C$ | — |
| $\int a^x dx$ | $\frac{a^x}{\ln a} + C$ | $a>0, a\neq 1$ |

### 2.2 三角函数

| 积分 | 结果 | | 积分 | 结果 |
|------|------|-|------|------|
| $\int \sin x dx$ | $-\cos x + C$ | | $\int \cos x dx$ | $\sin x + C$ |
| $\int \tan x dx$ | $-\ln|\cos x| + C$ | | $\int \cot x dx$ | $\ln|\sin x| + C$ |
| $\int \sec x dx$ | $\ln|\sec x + \tan x| + C$ | | $\int \csc x dx$ | $\ln|\csc x - \cot x| + C$ |
| $\int \sec^2 x dx$ | $\tan x + C$ | | $\int \csc^2 x dx$ | $-\cot x + C$ |
| $\int \sec x \tan x dx$ | $\sec x + C$ | | $\int \csc x \cot x dx$ | $-\csc x + C$ |

### 2.3 反三角函数

| 积分 | 结果 |
|------|------|
| $\int \frac{1}{\sqrt{1-x^2}}dx$ | $\arcsin x + C$ |
| $\int \frac{1}{1+x^2}dx$ | $\arctan x + C$ |
| $\int \frac{1}{\sqrt{a^2-x^2}}dx$ | $\arcsin\frac{x}{a} + C$ |
| $\int \frac{1}{a^2+x^2}dx$ | $\frac{1}{a}\arctan\frac{x}{a} + C$ |

## 三、第一类换元法（凑微分）

$$\int f(\varphi(x))\varphi'(x)dx = \int f(u)du \quad (u=\varphi(x))$$

**核心能力：识别被积表达式中的"导数因子"**。

### 3.1 经典凑微分模式

| 积分形式 | 凑法 | 结果 |
|---------|------|------|
| $\int f(ax+b)dx$ | $\frac{1}{a}\int f(u)du$ | 线性代换 |
| $\int f(\sin x)\cos x dx$ | $\int f(u)du, u=\sin x$ | — |
| $\int f(\cos x)\sin x dx$ | $-\int f(u)du, u=\cos x$ | — |
| $\int f(\ln x)\frac{1}{x}dx$ | $\int f(u)du, u=\ln x$ | — |
| $\int f(e^x)e^x dx$ | $\int f(u)du, u=e^x$ | — |
| $\int f(\tan x)\sec^2 x dx$ | $\int f(u)du, u=\tan x$ | — |
| $\int f(\arcsin x)\frac{1}{\sqrt{1-x^2}}dx$ | $\int f(u)du$ | — |
| $\int \frac{f'(x)}{f(x)}dx$ | $\ln|f(x)| + C$ | **常用！** |

### 3.2 凑微分例题

1. $\int \sin(3x+1)dx = -\frac{1}{3}\cos(3x+1) + C$

2. $\int x\sqrt{1+x^2}dx = \frac{1}{2}\int \sqrt{u}du = \frac{1}{3}(1+x^2)^{3/2} + C$, $u=1+x^2$

3. $\int \tan x dx = \int \frac{\sin x}{\cos x}dx = -\ln|\cos x| + C$

4. $\int \frac{e^x}{1+e^{2x}}dx = \int \frac{du}{1+u^2} = \arctan(e^x) + C$, $u=e^x$

## 四、第二类换元法（变量代换）

令 $x = \varphi(t)$：
$$\int f(x)dx = \int f(\varphi(t))\varphi'(t)dt$$

### 4.1 三角代换

| 被积函数含 | 令 | $dx$ 转化 | 还原用 |
|-----------|-----|-----------|--------|
| $\sqrt{a^2-x^2}$ | $x = a\sin t$ | $dx = a\cos t dt$ | $t = \arcsin\frac{x}{a}$ |
| $\sqrt{a^2+x^2}$ | $x = a\tan t$ | $dx = a\sec^2 t dt$ | $t = \arctan\frac{x}{a}$ |
| $\sqrt{x^2-a^2}$ | $x = a\sec t$ | $dx = a\sec t\tan t dt$ | $t = \text{arcsec}\frac{x}{a}$ |

### 4.2 倒代换

被积函数分母次数远高于分子时（如 $\int \frac{1}{x^n\sqrt{a^2\pm x^2}}dx$），令 $x = 1/t$。

### 4.3 根式代换

含 $\sqrt{ax+b}$ 或 $\sqrt{\frac{ax+b}{cx+d}}$ → 令整个根式为 $t$。

## 五、分部积分法

$$\int u dv = uv - \int v du$$

### 5.1 选 $u$ 的口诀

**"反对幂指三"**（按此顺序优先选 $u$）：
**反**三角函数 > **对**数函数 > **幂**函数 > **指**数函数 > **三**角函数

### 5.2 典型应用

| 类型 | $u$ | $dv$ | 结果要点 |
|------|-----|------|---------|
| $\int x^n e^{ax}dx$ | $x^n$ | $e^{ax}dx$ | $n$ 次分部积分消去多项式 |
| $\int x^n \sin ax dx$ | $x^n$ | $\sin ax dx$ | 同上 |
| $\int x^n \ln x dx$ | $\ln x$ | $x^n dx$ | $du = \frac{1}{x}dx$，降幂 |
| $\int e^{ax}\sin bx dx$ | 任意 | 任意 | 两次分部积分解方程 |
| $\int \arcsin x dx$ | $\arcsin x$ | $dx$ | $= x\arcsin x + \sqrt{1-x^2} + C$ |
| $\int \arctan x dx$ | $\arctan x$ | $dx$ | $= x\arctan x - \frac{1}{2}\ln(1+x^2) + C$ |

### 5.3 递推公式型

$\int \sin^n x dx = -\frac{1}{n}\sin^{n-1}x\cos x + \frac{n-1}{n}\int \sin^{n-2}x dx$

$\int \cos^n x dx = \frac{1}{n}\cos^{n-1}x\sin x + \frac{n-1}{n}\int \cos^{n-2}x dx$

## 六、有理函数积分

### 6.1 部分分式分解

$$\frac{P(x)}{Q(x)} = \sum \frac{A_i}{(x-a)^i} + \sum \frac{B_j x + C_j}{(x^2+px+q)^j}$$

分母因式分解为一次和二次不可约因子，写成部分分式之和。

### 6.2 四类基本分式的积分

1. $\int \frac{A}{x-a}dx = A\ln|x-a| + C$
2. $\int \frac{A}{(x-a)^n}dx = \frac{A}{1-n}\frac{1}{(x-a)^{n-1}} + C$ ($n>1$)
3. $\int \frac{Ax+B}{x^2+px+q}dx$：分子凑成分母导数 + 常数 + 配方成 $\arctan$
4. $\int \frac{Ax+B}{(x^2+px+q)^n}dx$：凑微分 + 递推

## 七、三角函数有理式的积分

### 7.1 万能代换

令 $t = \tan\frac{x}{2}$：
$$\sin x = \frac{2t}{1+t^2}, \quad \cos x = \frac{1-t^2}{1+t^2}, \quad dx = \frac{2}{1+t^2}dt$$

适用于一切 $\int R(\sin x, \cos x)dx$ 型积分，但有时计算量大。

### 7.2 $\sin^m x \cos^n x$ 型的策略

| $m,n$ 情况 | 策略 |
|-----------|------|
| $m$ 奇 | 拿出一个 $\sin x$ 凑 $d(-\cos x)$ |
| $n$ 奇 | 拿出一个 $\cos x$ 凑 $d(\sin x)$ |
| $m,n$ 都为偶 | 用倍角公式降次：$\sin^2x=\frac{1-\cos2x}{2}, \cos^2x=\frac{1+\cos2x}{2}$ |

## 八、易错点

1. **$\int \frac{1}{x}dx = \ln|x| + C$**：是 $|x|$ 不是 $x$，不能忘绝对值
2. **凑微分后注意系数调整**：$\int \cos 2x dx = \frac{1}{2}\sin 2x + C$
3. **分部积分 $u$ 选错会越积越复杂**
4. **不定积分必须 $+C$**：忘记 $C$ 在考试中是严重扣分项
