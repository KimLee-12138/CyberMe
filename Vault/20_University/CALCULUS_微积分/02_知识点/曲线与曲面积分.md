# 曲线积分与曲面积分

> 核心：三大公式（格林、高斯、斯托克斯）将边界积分转化为内部积分。这是 N-L 公式的高维推广。

## 一、第一类曲线积分（对弧长）

$$\int_L f ds = \int_\alpha^\beta f(x(t),y(t))\sqrt{[x'(t)]^2+[y'(t)]^2}dt$$

无方向性，物理意义为线密度积分。$y=y(x)$ 时 $ds=\sqrt{1+[y']^2}dx$。

## 二、第二类曲线积分（对坐标）

$$\int_L Pdx+Qdy = \int_\alpha^\beta [P x'(t)+Q y'(t)]dt$$

**有方向性**：反向积分变号。物理意义：力 $\vec{F}=(P,Q)$ 做功。

## 三、格林公式

$$\oint_L Pdx+Qdy = \iint_D \left(\frac{\partial Q}{\partial x} - \frac{\partial P}{\partial y}\right) dxdy$$

条件：$L$ 封闭正向（逆时针），$P,Q$ 在 $D$ 上有一阶连续偏导。

### 3.1 面积公式
$$A = \frac{1}{2}\oint_L xdy - ydx$$

例：椭圆 $x=a\cos t, y=b\sin t \to A=\pi ab$

### 3.2 路径无关条件

$\frac{\partial Q}{\partial x} = \frac{\partial P}{\partial y}$ 在单连通域内恒成立 → 积分与路径无关 → 存在势函数。

**求原函数**：$u(x,y)=\int_{x_0}^x P(t,y_0)dt+\int_{y_0}^y Q(x,t)dt$

### 3.3 非封闭曲线处理
补线段封闭化 → 格林公式 → 减补线积分。

## 四、第一类曲面积分

曲面 $z=z(x,y)$：$\iint_\Sigma f dS = \iint_D f(x,y,z)\sqrt{1+z_x^2+z_y^2}dxdy$

## 五、高斯公式（散度定理）

$$\oiint_\Sigma \vec{F}\cdot d\vec{S} = \iiint_\Omega \text{div}\vec{F} dV$$

**散度**：$\text{div}\vec{F} = P_x+Q_y+R_z$。条件：$\Sigma$ 封闭曲面**外侧**。

非封闭曲面 → 补面 + 高斯 - 补面积分。

## 六、斯托克斯公式

$$\oint_L \vec{F}\cdot d\vec{r} = \iint_\Sigma \text{rot}\vec{F}\cdot d\vec{S}$$

**旋度**：$\text{rot}\vec{F} = \nabla\times\vec{F} = (R_y-Q_z, P_z-R_x, Q_x-P_y)$

**格林公式是斯托克斯在平面上的特例**（$R=0$）。

## 七、三大公式关系

```
N-L公式(1D): ∫f' = f(b)-f(a)
    ↓
格林(2D): ∮ = ∬(Qx-Py)  →  高斯(3D封闭曲面→三重积分)
                           斯托克斯(3D封闭曲线→曲面积分)
```

## 八、策略速查

| 场景 | 方法 |
|------|------|
| 封闭曲线线积分 | 格林公式 |
| 非封闭曲线 | 补线+格林 / 直接参数化 |
| 封闭曲面积分 | 高斯公式 |
| 空间封闭曲线 | 斯托克斯 |
| $Q_x=P_y$ | 路径无关→换路径 |

## 九、易错点

1. 第二类线积分有方向（反向变号），第一类无方向
2. 格林公式右端是 $Q_x-P_y$，顺序不能反
3. 高斯公式必须外侧，内侧加负号
4. 补线/补面后务必减掉补充部分的积分
