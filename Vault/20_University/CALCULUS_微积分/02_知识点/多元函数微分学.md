# 多元函数微分学

> 一句话：将一元微积分推广到多维空间。核心是偏导数、全微分、方向导数与极值。

## 一、基本概念

### 1.1 二元函数的极限

$$\lim_{(x,y) \to (x_0,y_0)} f(x,y) = A$$

**注意**：$(x,y)$ 沿**所有路径**趋近时极限都相同才存在。沿不同路径得不同极限→极限不存在。

**典型反例**：$f(x,y) = \frac{x^2-y^2}{x^2+y^2}$ 在 $(0,0)$ 处沿 $y=0$ 趋近得 1，沿 $x=0$ 趋近得 -1 → 极限不存在。

### 1.2 连续

$f(x,y)$ 在 $(x_0,y_0)$ 连续 ⟺ 极限存在且等于函数值。

**注意**：偏导数存在 ≠ 连续（与一元不同！）。

## 二、偏导数

### 2.1 定义

$$\frac{\partial f}{\partial x}\bigg|_{(x_0,y_0)} = \lim_{\Delta x \to 0} \frac{f(x_0+\Delta x, y_0) - f(x_0,y_0)}{\Delta x}$$

求 $\frac{\partial f}{\partial x}$ 时，将 $y$ **视为常数**，按一元函数求导。

### 2.2 二阶偏导数

- $\frac{\partial^2 f}{\partial x^2}$：对 $x$ 求两次偏导
- $\frac{\partial^2 f}{\partial y^2}$：对 $y$ 求两次偏导
- $\frac{\partial^2 f}{\partial x\partial y}$：先对 $x$ 再对 $y$
- $\frac{\partial^2 f}{\partial y\partial x}$：先对 $y$ 再对 $x$

**克莱罗定理**：若 $f_{xy}$ 和 $f_{yx}$ 连续，则：
$$\frac{\partial^2 f}{\partial x\partial y} = \frac{\partial^2 f}{\partial y\partial x}$$

即**求导次序可交换**。

## 三、全微分

### 3.1 定义

若 $\Delta z = f(x_0+\Delta x, y_0+\Delta y) - f(x_0,y_0)$ 可表示为：
$$\Delta z = A\Delta x + B\Delta y + o(\sqrt{\Delta x^2+\Delta y^2})$$

则 $f$ 在 $(x_0,y_0)$ 处**可微**，全微分为：
$$dz = \frac{\partial f}{\partial x}dx + \frac{\partial f}{\partial y}dy$$

### 3.2 可微的判定

**充分条件**：偏导数 $f_x, f_y$ 在点 $(x_0,y_0)$ 处**连续**。

**必要条件**：$f$ 可微 ⇒ $f$ 连续，且偏导数存在。

**注意**：偏导数存在 ≠ 可微（反例：$f(x,y)=\frac{xy}{x^2+y^2}$ 在原点偏导数存在但不可微）。

### 3.3 全微分的应用

**近似计算**：$f(x+\Delta x, y+\Delta y) \approx f(x,y) + f_x\Delta x + f_y\Delta y$

例：计算 $\sqrt{1.02^2 + 1.99^2}$ 的近似值。（设 $f(x,y)=\sqrt{x^2+y^2}$，取 $(x_0,y_0)=(1,2)$）

## 四、链式法则

### 4.1 多元复合函数求导

$z = f(u,v), u = u(x,y), v = v(x,y)$：

$$\frac{\partial z}{\partial x} = \frac{\partial f}{\partial u}\frac{\partial u}{\partial x} + \frac{\partial f}{\partial v}\frac{\partial v}{\partial x}$$

$$\frac{\partial z}{\partial y} = \frac{\partial f}{\partial u}\frac{\partial u}{\partial y} + \frac{\partial f}{\partial v}\frac{\partial v}{\partial y}$$

**方法**：画变量树形图，从 $z$ 出发，经过所有到达目标变量的路径，每条路径上连乘。

### 4.2 隐函数求导公式

**一个方程 $F(x,y)=0$** 确定 $y=y(x)$：
$$\frac{dy}{dx} = -\frac{F_x}{F_y}$$

**一个方程 $F(x,y,z)=0$** 确定 $z=z(x,y)$：
$$\frac{\partial z}{\partial x} = -\frac{F_x}{F_z}, \quad \frac{\partial z}{\partial y} = -\frac{F_y}{F_z}$$

**方程组隐函数**（雅可比行列式）：若 $\begin{cases}F(x,y,u,v)=0 \\ G(x,y,u,v)=0\end{cases}$ 确定 $u=u(x,y), v=v(x,y)$，使用克莱姆法则。

## 五、方向导数与梯度

### 5.1 方向导数

沿单位向量 $\vec{l} = (\cos\alpha, \cos\beta, \cos\gamma)$ 的方向导数：
$$\frac{\partial f}{\partial \vec{l}} = f_x\cos\alpha + f_y\cos\beta + f_z\cos\gamma = \nabla f \cdot \vec{l}$$

### 5.2 梯度

$$\text{grad} f = \nabla f = \left(\frac{\partial f}{\partial x}, \frac{\partial f}{\partial y}, \frac{\partial f}{\partial z}\right)$$

**梯度的性质**：
1. **梯度方向 = 函数值增长最快的方向**
2. **$|\nabla f|$ = 最大的方向导数**
3. **梯度 ⟂ 等值面**（$f(x,y,z)=C$ 的法向量即 $\nabla f$）

### 5.3 物理意义

- 温度场中，梯度指向温度升高最快的方向
- 势能场中，负梯度 = 力（保守力 = $-\nabla U$）

## 六、空间曲面的切平面与法线

### 6.1 曲面 $F(x,y,z) = 0$

在点 $P_0(x_0,y_0,z_0)$ 处：
- **法向量**：$\vec{n} = (F_x, F_y, F_z)\big|_{P_0}$
- **切平面**：$F_x(P_0)(x-x_0) + F_y(P_0)(y-y_0) + F_z(P_0)(z-z_0) = 0$
- **法线**：$\frac{x-x_0}{F_x} = \frac{y-y_0}{F_y} = \frac{z-z_0}{F_z}$

### 6.2 曲面 $z = f(x,y)$

- 法向量：$\vec{n} = (-f_x, -f_y, 1)$
- 切平面：$z - z_0 = f_x(x-x_0) + f_y(y-y_0)$

## 七、多元函数的极值

### 7.1 无条件极值

**必要条件**：驻点 → $\nabla f = \vec{0}$（$f_x=0, f_y=0$）

**充分条件**（二阶判别）：

计算 $A = f_{xx}, B = f_{xy}, C = f_{yy}$，则 **Hessian 判别式**：
$$H = AC - B^2 = f_{xx}f_{yy} - (f_{xy})^2$$

| $H(P_0)$ | $A$ | 结论 |
|----------|-----|------|
| $> 0$ | $> 0$ | **极小值** |
| $> 0$ | $< 0$ | **极大值** |
| $< 0$ | — | 不是极值（**鞍点**） |
| $= 0$ | — | 无法判定，需用定义 |

### 7.2 条件极值 — 拉格朗日乘数法

问题：求 $z = f(x,y)$ 在约束 $\varphi(x,y) = 0$ 下的极值。

**步骤**：
1. 构造拉格朗日函数：$L(x,y,\lambda) = f(x,y) + \lambda\varphi(x,y)$
2. 解方程组：
$$\begin{cases} L_x = f_x + \lambda\varphi_x = 0 \\ L_y = f_y + \lambda\varphi_y = 0 \\ L_\lambda = \varphi(x,y) = 0 \end{cases}$$
3. 比较各驻点的函数值，得极值。

**多个约束**：$m$ 个约束 = $m$ 个 $\lambda$ 乘子。

## 八、典型例题思路

1. **求极限**：判断 $(x,y)\to(0,0)$ 时沿 $y=kx$ 路径趋近 → 极限只与 $k$ 无关才可能存在
2. **证明偏导存在但不可微**：$f_x(0,0)$ 和 $f_y(0,0)$ 可由定义求出，但 $\Delta z - f_x\Delta x - f_y\Delta y$ 不是 $o(\rho)$
3. **条件极值**：拉格朗日法适用于等量约束；不等式约束需用 KKT 条件

## 九、易错点

1. **偏导存在 ≠ 连续**：这与一元函数完全不同
2. **偏导存在 ≠ 可微**：偏导**连续**才是可微的充分条件
3. **链式法则不能遗漏任何中间变量**：画树形图检查
4. **拉格朗日乘数法得到的只是候选点**：还需比较边界值（如有）
5. **Hessian 判别 $H=0$ 时不能下结论**：需回到定义判断
