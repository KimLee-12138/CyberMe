# 重积分

> 一句话：定积分从一维推广到二维和三维。核心是把重积分化为累次积分来计算。

## 一、二重积分

### 1.1 定义

$$\iint_D f(x,y)d\sigma = \lim_{\lambda \to 0} \sum_{i=1}^n f(\xi_i, \eta_i)\Delta\sigma_i$$

**几何意义**（$f \ge 0$）：以 $D$ 为底、$z=f(x,y)$ 为顶的曲顶柱体体积。

**物理意义**：$f$ 为面密度时，$\iint_D f d\sigma$ = 薄板质量。

### 1.2 直角坐标下的累次积分

**X 型区域**（先 $y$ 后 $x$）：
$$D: a \le x \le b, \quad \varphi_1(x) \le y \le \varphi_2(x)$$
$$\iint_D f d\sigma = \int_a^b \left[ \int_{\varphi_1(x)}^{\varphi_2(x)} f(x,y) dy \right] dx$$

**Y 型区域**（先 $x$ 后 $y$）：
$$D: c \le y \le d, \quad \psi_1(y) \le x \le \psi_2(y)$$
$$\iint_D f d\sigma = \int_c^d \left[ \int_{\psi_1(y)}^{\psi_2(y)} f(x,y) dx \right] dy$$

**选择顺序口诀**：后积分的变量范围是常数。哪个方向的"投影"更规则就选哪个。

### 1.3 极坐标变换

$$x = r\cos\theta, \quad y = r\sin\theta, \quad d\sigma = \mathbf{r}\,dr d\theta$$

**关键**：面积微元多了一个 **$r$**，这是极坐标变换中最容易漏掉的！

$$\iint_D f(x,y)d\sigma = \int_\alpha^\beta d\theta \int_{r_1(\theta)}^{r_2(\theta)} f(r\cos\theta, r\sin\theta) \cdot \mathbf{r}\, dr$$

**适用场景**：
- 圆形区域 $x^2+y^2 \le R^2$ → $\theta \in [0,2\pi], r \in [0,R]$
- 被积函数含 $x^2+y^2$（化为 $r^2$）
- 扇形、环形区域

### 1.4 二重积分的应用

| 应用 | 公式 |
|------|------|
| 平面面积 | $A = \iint_D d\sigma$ |
| 体积（$f \ge 0$） | $V = \iint_D f(x,y)d\sigma$ |
| 曲面 $z=f(x,y)$ 的面积 | $A = \iint_D \sqrt{1+f_x^2+f_y^2}dxdy$ |
| 质量（面密度 $\rho$） | $m = \iint_D \rho(x,y)d\sigma$ |
| 质心 | $\bar{x} = \frac{\iint x\rho d\sigma}{\iint \rho d\sigma}$ |

## 二、三重积分

### 2.1 直角坐标

$$\iiint_\Omega f(x,y,z)dV = \iint_D \left[ \int_{z_1(x,y)}^{z_2(x,y)} f dz \right] dxdy$$

先定 $z$ 的范围（上下曲面），再将投影区域转化为二重积分。

### 2.2 柱坐标

$$x = r\cos\theta, \quad y = r\sin\theta, \quad z = z$$
$$dV = \mathbf{r}\,dr d\theta dz$$

**适用**：积分区域为圆柱、圆锥、旋转抛物面等。

例：圆柱体 $x^2+y^2\le R^2, 0\le z\le H$ → $0\le\theta\le 2\pi, 0\le r\le R, 0\le z\le H$

### 2.3 球坐标

$$x = \rho\sin\varphi\cos\theta,\quad y = \rho\sin\varphi\sin\theta,\quad z = \rho\cos\varphi$$
$$dV = \mathbf{\rho^2\sin\varphi}\,d\rho d\varphi d\theta$$

- $\rho$：原点到点的距离（$0 \le \rho$）
- $\varphi$：与 $z$ 轴正向的夹角（$0 \le \varphi \le \pi$）
- $\theta$：投影到 $xy$ 平面与 $x$ 轴的夹角（$0 \le \theta \le 2\pi$）

**适用**：球体或球的一部分、锥体。

**注意**：不同教材对 $\varphi$ 和 $\theta$ 的定义可能不同，考试要确认符号！

## 三、重积分的一般换元法

### 3.1 雅可比行列式

$u = u(x,y), v = v(x,y)$，反过来 $x = x(u,v), y = y(u,v)$：

$$\iint_D f(x,y)dxdy = \iint_{D'} f(x(u,v), y(u,v)) \cdot \left|\frac{\partial(x,y)}{\partial(u,v)}\right| dudv$$

其中：
$$J = \frac{\partial(x,y)}{\partial(u,v)} = \begin{vmatrix} x_u & x_v \\ y_u & y_v \end{vmatrix}$$

**极坐标的 $J = r$**：$x=r\cos\theta, y=r\sin\theta \Rightarrow J = \begin{vmatrix} \cos\theta & -r\sin\theta \\ \sin\theta & r\cos\theta \end{vmatrix} = r$

## 四、计算技巧

### 4.1 利用对称性

**二重积分**：
- 积分区域 $D$ 关于 $x$ 轴对称：若 $f$ 关于 $y$ 为奇函数 → 积分为 0；偶函数 → 2 倍上半区域
- 积分区域 $D$ 关于 $y=x$ 对称：$\iint_D f(x,y)d\sigma = \iint_D f(y,x)d\sigma$

**三重积分**同样适用奇偶性和对称性。

### 4.2 交换积分次序

当先 $y$ 后 $x$ 遇到困难（如 $\int e^{y^2}dy$ 无初等原函数），交换为先 $x$ 后 $y$。画出积分区域是关键。

### 4.3 割补法

将复杂区域拆分成若干简单区域之和或差。

## 五、易错点

1. **极坐标必须乘 $r$**：$d\sigma = r dr d\theta$，忘乘等于白做
2. **球坐标 $dV = \rho^2\sin\varphi d\rho d\varphi d\theta$**：$\rho^2$ 和 $\sin\varphi$ 都不能漏
3. **累次积分的上下限**：外层变量的范围必须是常数
4. **对称性只能用于积分区域 + 被积函数都满足时**
