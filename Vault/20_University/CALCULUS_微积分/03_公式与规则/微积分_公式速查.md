# 微积分公式速查

> 考前快速浏览。按模块分类。

## 一、基本导数

| $f(x)$ | $f'(x)$ | | $f(x)$ | $f'(x)$ |
|--------|---------|-|--------|---------|
| $C$ | $0$ | | $x^n$ | $nx^{n-1}$ |
| $e^x$ | $e^x$ | | $\ln x$ | $1/x$ |
| $\sin x$ | $\cos x$ | | $\cos x$ | $-\sin x$ |
| $\tan x$ | $\sec^2 x$ | | $\arcsin x$ | $\frac{1}{\sqrt{1-x^2}}$ |
| $\arctan x$ | $\frac{1}{1+x^2}$ |

## 二、基本积分

| . | . |
|----|----|
| $\int x^n dx = \frac{x^{n+1}}{n+1}+C$ | $\int \frac{1}{x}dx = \ln|x|+C$ |
| $\int e^x dx = e^x+C$ | $\int \sin x dx = -\cos x+C$ |
| $\int \cos x dx = \sin x+C$ | $\int \sec^2 x dx = \tan x+C$ |
| $\int \frac{1}{1+x^2}dx = \arctan x+C$ | $\int \frac{1}{\sqrt{1-x^2}}dx = \arcsin x+C$ |

## 三、一阶线性微分方程

$$y' + Py = Q \Rightarrow y = e^{-\int Pdx}\left[\int Q e^{\int Pdx}dx + C\right]$$

## 四、二阶常系数齐次方程

特征方程 $r^2 + pr + q = 0$：
- $\Delta>0$：$y=C_1e^{r_1x}+C_2e^{r_2x}$
- $\Delta=0$：$y=(C_1+C_2x)e^{rx}$
- $\Delta<0$：$y=e^{\alpha x}(C_1\cos\beta x+C_2\sin\beta x)$

## 五、重要级数

- 几何：$\sum aq^{n-1}$，$|q|<1$ 收敛，和 $S=\frac{a}{1-q}$
- **p-级数**：$\sum\frac{1}{n^p}$，**$p>1$ 收敛，$p\le 1$ 发散**
- 调和级数 $\sum\frac{1}{n}$ 发散

## 六、麦克劳林展开

- $e^x = \sum_{n=0}^{\infty}\frac{x^n}{n!}$
- $\sin x = \sum_{n=0}^{\infty}\frac{(-1)^n x^{2n+1}}{(2n+1)!}$
- $\frac{1}{1-x} = \sum_{n=0}^{\infty} x^n$ ($R=1$)
- $\ln(1+x) = \sum_{n=1}^{\infty}\frac{(-1)^{n-1}x^n}{n}$ ($R=1$)

## 七、格林/高斯/斯托克斯

- **格林**：$\oint_L Pdx+Qdy = \iint_D(Q_x-P_y)dxdy$
- **高斯**：$\oiint_\Sigma \vec{F}\cdot d\vec{S} = \iiint_\Omega \text{div}\vec{F} dV$
- **斯托克斯**：$\oint_L \vec{F}\cdot d\vec{r} = \iint_\Sigma \text{rot}\vec{F}\cdot d\vec{S}$

## 八、坐标变换雅可比

- 极坐标：$dxdy = r dr d\theta$
- 柱坐标：$dV = r dr d\theta dz$
- 球坐标：$dV = r^2\sin\varphi dr d\varphi d\theta$
