# 微积分错题记录

---

## #1：不定积分忘加 C
$\int \sin x dx = -\cos x + C$（不是 $-\cos x$）
□ □ □

## #2：定积分换元忘换上下限
$\int_0^{\pi/2} \sin^2x\cos x dx$：令 $u=\sin x$，上下限变为 0 和 1
□ □ □

## #3：极坐标忘乘 r
$dxdy = r dr d\theta$，必须乘 $r$！
□ □ □

## #4：一阶线性公式符号
$y'+Py=Q$，$P$ 前是 **+** 号。$y = e^{-\int P}(\int Q e^{\int P} + C)$
□ □ □

## #5：复根通解忘 $e^{\alpha x}$
$r=\alpha\pm i\beta \Rightarrow y = e^{\alpha x}(C_1\cos\beta x + C_2\sin\beta x)$
□ □ □

## #6：p-级数判敛
$\sum\frac{1}{n^p}$：$p>1$ 收敛，$p\le 1$ 发散（不要背反）
□ □ □

## #7：通项趋于 0 是必要非充分
$\sum\frac{1}{n}$ 通项趋于 0 但发散
□ □ □

## #8：格林公式要求封闭曲线
不封闭需"补线"。$Q_x$ 和 $P_y$ 的顺序不能反
□ □ □

## #9：拉格朗日乘数法方程个数
有 $n$ 个变量 + $m$ 个约束 = $n+m$ 个方程
□ □ □
