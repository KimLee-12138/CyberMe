# 电磁学基础

## 一句话解释

静止电荷产生静电场（库仑定律），运动电荷（电流）产生磁场（毕奥-萨伐尔定律）。电磁感应的核心：变化的磁场产生电场（法拉第定律），变化的电场产生磁场（位移电流）——麦克斯韦用四个方程统一了电磁学，并预言了电磁波的存在。

## 核心原理

### 静电场

**库仑定律**：F = k q₁q₂/r²，方向沿连线。k = 1/(4πε₀) ≈ 9×10⁹ N·m²/C²

**电场强度**：E = F/q₀（单位正电荷在该点受到的力）

**高斯定理**：∮_S E·dS = q_enc/ε₀（闭合面的电通量 = 内部净电荷/ε₀）

巧妙选择高斯面（利用对称性）可以快速求电场：
- 点电荷 → 球面：E·4πr² = q/ε₀ → E=kq/r²
- 无限长带电线 → 柱面：E·2πrL = λL/ε₀ → E=λ/(2πε₀r)
- 无限大带电平面 → 柱面：E·2A = σA/ε₀ → E=σ/(2ε₀)

**电势**：V_a - V_b = ∫_a^b E·dl（电场强度从 a 到 b 的线积分）

点电荷电势：V = kq/r

### 静磁场

**毕奥-萨伐尔定律**：dB = (μ₀/4π)·(Idl × r̂)/r²

用来计算任意形状载流导线产生的磁场。

**安培环路定理**：∮_L B·dl = μ₀ I_enc（闭合回路的 B 的环量 = μ₀ × 穿过回路的电流）

**洛伦兹力**：F = qv × B（磁场对运动电荷的力）
- 力垂直于 v 和 B 所在平面
- 洛伦兹力不做功（因为力始终垂直于速度方向）
- 带电粒子在均匀磁场中做**匀速圆周运动**：qvB=mv²/R → R=mv/(qB)

### 电磁感应

**法拉第电磁感应定律**：ε = -dΦ_B/dt

感应电动势 = 磁通量变化率的负值。负号来自**楞次定律**。

**楞次定律**：感应电流的方向总是使其产生的磁场**阻碍**引起感应的磁通量变化。

- 磁铁靠近线圈 → 线圈产生排斥磁铁的磁场（阻碍靠近）
- 磁铁远离线圈 → 线圈产生吸引磁铁的磁场（阻碍远离）

**动生电动势**：ε = Blv（导体在磁场中切割磁力线）

### 麦克斯韦方程组（积分形式）

1. **高斯定律（电场）**：∮E·dS = q_enc/ε₀（电荷产生电场）
2. **高斯定律（磁场）**：∮B·dS = 0（无磁单极子——磁感线闭合！）
3. **法拉第定律**：∮E·dl = -dΦ_B/dt（变化的磁场产生电场）
4. **安培-麦克斯韦定律**：∮B·dl = μ₀(I + ε₀·dΦ_E/dt)（电流+**变化的电场**产生磁场）

麦克斯韦的贡献：在第4式加入位移电流项 ε₀·dΦ_E/dt，预言了**电磁波**。

## 规则或公式

| 公式 | 说明 |
|------|------|
| F = kq₁q₂/r² | 库仑定律 |
| ∮E·dS = q/ε₀ | 高斯定理 |
| F = qv×B | 洛伦兹力 |
| ε = -dΦ_B/dt | 法拉第电磁感应 |
| R = mv/(qB) | 带电粒子圆周运动半径 |

## 容易出错的地方

- **电场和磁场方向判断**：右手定则用于(1)电流→磁场方向，(2)v×B→洛伦兹力方向。左手定则不要混淆
- 高斯面的选择要有对称性：球对称→球面，柱对称→柱面，面对称→柱面
- 课程导航：[[../00_大学物理_课程MOC|大学物理 MOC]]


- 感应电动势的**负号**（楞次定律）不要丢：它告诉你感应电流的方向
- 洛伦兹力**不做功**——只改变速度方向不改变速度大小（速率不变）
