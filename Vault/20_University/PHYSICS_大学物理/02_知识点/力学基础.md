# 力学基础

## 一句话解释

牛顿三大定律是经典力学的基石：惯性定律（不受力则速度不变）、F=ma（力决定加速度）、作用力与反作用力。在此基础上发展出三大守恒律（能量守恒、动量守恒、角动量守恒），它们比牛顿定律更基本——在相对论和量子力学中仍然成立。

## 核心原理

### 牛顿三大定律

**第一定律（惯性定律）**：不受外力（或合外力为零）时，物体保持静止或匀速直线运动。

定义了惯性参考系——牛顿定律成立的参考系。

**第二定律**：**F = ma = m·dv/dt = dp/dt**（动量变化率 = 合外力）

这是经典力学的核心方程。F=ma 是矢量方程（三维中对应三个分量方程）。

**第三定律**：F_{AB} = -F_{BA}（A 对 B 的力 = B 对 A 的力的反作用力）

同时产生、同时消失、同性质、共线、反向、作用在不同物体上。

### 常见力

- **重力**：G = mg, 方向竖直向下
- **弹力**（胡克定律）：F = -kx（弹簧恢复力与形变成正比，方向指向平衡位置）
- **摩擦力**：静摩擦力 0 ≤ f_s ≤ μ_s N（物体未动时的摩擦力）；滑动摩擦力 f_k = μ_k N（物体滑动时）
- **万有引力**：F = GmM/r²

### 功与能

**功**：W = ∫F·dr（力在位移上的积累）
- 恒力直线：W = F·s·cosθ
- 保守力（重力/弹力/万有引力/静电力）：做功与路径无关，只与起点终点有关

**动能**：E_k = ½mv²

**动能定理**：W_合 = ΔE_k（合外力做功 = 动能变化量）

**势能**：保守力对应的能量
- 重力势能：E_p = mgh（以某参考面为零势能面）
- 弹性势能：E_p = ½kx²（以弹簧自然长度处为零势能点）
- 万有引力势能：E_p = -GmM/r（无穷远为零势能点）

**机械能守恒**：只有保守力做功时，E_k + E_p = 常数

**功能原理**：W_外 + W_非保守内力 = ΔE_机械（外力+非保守内力做功 = 机械能变化）

### 动量与冲量

**动量**：p = mv（矢量）

**冲量**：I = ∫F dt（力在时间上的积累）

**动量定理**：I = Δp（冲量 = 动量变化量）——F·Δt = mv - mv₀

**动量守恒**：系统不受外力（或合外力为0）时，总动量守恒。这是自然界最基本的守恒律之一。

**碰撞问题**：
- 完全弹性碰撞：动量守恒 + 动能守恒
- 完全非弹性碰撞：动量守恒，动能损失最大（粘在一起）
- 恢复系数 e = (v₂'-v₁')/(v₁-v₂)：e=1 完全弹性，e=0 完全非弹性

### 刚体转动

**角速度 ω 与线速度 v**：v = ω×r, |v|=ωr

**转动惯量 I**（衡量物体抵抗转动加速的能力）：
- 质点：I = mr²
- 细棒绕端点：I = (1/3)mL²
- 细棒绕中心：I = (1/12)mL²
- 圆盘/圆柱绕中心轴：I = ½mR²
- 球体绕直径：I = (2/5)mR²

**转动定律**：τ = Iα（力矩 = 转动惯量 × 角加速度）——转动版的 F=ma

**角动量**：L = Iω

**角动量守恒**：合外力矩为0时，角动量守恒。花样滑冰运动员收臂→I减小→ω增大→转得更快。

## 规则或公式

| 公式 | 说明 |
|------|------|
| F = ma = dp/dt | 牛顿第二定律 |
| W = Fs cosθ | 恒力做功 |
| E_k = ½mv² | 动能 |
| p = mv | 动量 |
| τ = Iα | 转动定律 |

## 容易出错的地方

- **动能定理中的 W 是合外力做功**，不是某个力的功
- **动量守恒条件**：合外力=0。碰撞时内力极大，短暂外力可忽略→近似守恒
- 课程导航：[[../00_大学物理_课程MOC|大学物理 MOC]]


- 机械能守恒条件：**只有保守力做功**。有摩擦力就不守恒（机械能转化为内能）
- 动量是矢量——碰撞问题要选正方向，动量分量分别守恒
