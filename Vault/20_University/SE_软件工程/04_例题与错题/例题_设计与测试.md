# 例题_设计与测试

## 例题1：内聚耦合

**题目**：以下设计有什么问题？模块A直接访问模块B的内部变量，模块C通过函数参数传递与模块D通信。

**答案**：A→B：内容耦合（最差），应改为通过接口访问。C→D：数据耦合（较好）。应遵循高内聚低耦合原则。


## 例题2：等价类划分

**题目**：输入年龄(0-150)，用等价类划分法设计测试用例。

**答案**：有效等价类：[0,150]→选75。无效等价类：<0→选-1；>150→选151；非数字→选"abc"。共4个用例。


## 例题3：白盒测试

**题目**：以下代码需要多少个测试用例达到判定覆盖？if(x>0 && y>0) return 1; else return 0;

**答案**：判定覆盖需2个用例：①x>0,y>0(true分支)；②x>0,y≤0或x≤0(任意，false分支)。
