# 变量

1.寄存器变量（register）

- 使用频率较高。

2.静态变量（static）

- 只能被初始化一次，变量的值可以保存到下次进入函数，是函数具有记忆功能。

3.全局变量（extern）

- 使用extern声明后，可以在变量的定义前调用

- 可以在同一个项目进展，跨文件操作

  

  静态局部变量和全局变量自动初始化位0
