# 面向对象

## 一、三大特性

| 特性 | 说明 |
|------|------|
| **封装** | 隐藏内部实现(private/protected/public) |
| **继承** | 子类复用父类代码，扩展功能 |
| **多态** | 同一接口不同实现(虚函数/覆写) |

## 二、类与对象

类=模板，对象=实例。构造/析构函数。

## 三、抽象类与接口

纯虚函数(C++：`virtual void f()=0`)。定义契约。

## 四、设计原则(SOLID)

单一职责、开闭原则、里氏替换、接口隔离、依赖反转。
