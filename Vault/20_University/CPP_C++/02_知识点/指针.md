# 指针

## 一句话解释
指针存储另一个变量的内存地址。`int* p=&a;`定义指针，`*p`解引用访问指向的值。指针和数组密切相关(a[i]等价于*(a+i))。C++的指针比Java的引用更底层、更灵活、也更危险。

## 核心原理
**指针基础**：&取地址，*解引用。指针本身也有地址(&p)。指针变量占用内存(32位系统4字节，64位系统8字节)。

**指针与数组**：数组名a是常量指针(指向首元素，不能修改a的指向)。a[i]≡*(a+i)。`int arr[5]; int* p=arr;` p指向arr[0]。二维数组：`int a[3][4]; int (*p)[4]=a;` p是指向"4个int的数组"的指针。

**指针与函数**：传指针实现"引用"效果(`void swap(int*a,int*b)`)。函数指针：`int (*fp)(int,int)=&add;`。返回指针的函数：`int* f(){...}`(不能返回局部变量地址！)。

**const与指针(从右往左读)**：const int* p(指向常量的指针，不能通过p修改值)；int* const p(常量指针，p不能改指向)；const int* const p(都不能改)。

**多级指针**：int** pp=&p(指向指针的指针)。用于二维数组的动态分配、函数中修改指针本身。

## 完整例子
```cpp
int a=10; int* p=&a; *p=20; // a变为20
int arr[]={1,2,3,4,5};
for(int* p=arr; p!=arr+5; p++) cout<<*p<<" "; // 指针遍历数组
// 动态分配
int* dyn = new int[100];
delete[] dyn; // 必须用delete[]匹配new[]
```

## 容易出错的地方
- 野指针(未初始化/已释放)→[[空指针与野指针]]
- 指针运算的单位是sizeof(T)(p+1跳sizeof(T)字节)
- new和delete必须配对，new[]配delete[]

- 函数返回局部变量地址→悬空指针
