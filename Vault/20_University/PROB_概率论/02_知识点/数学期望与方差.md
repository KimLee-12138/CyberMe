# 数学期望与方差

## 一句话解释

期望是随机变量的"平均值"——以概率为权的加权平均。方差衡量随机变量偏离期望的程度。协方差和相关系数刻画两个随机变量之间的线性相关关系。

## 核心原理

### 数学期望 E(X)

**离散型**：E(X) = Σ xₖ·pₖ

**连续型**：E(X) = ∫_{-∞}^{+∞} x·f(x)dx（要求绝对收敛，即 ∫|x|f(x)dx < ∞）

**随机变量函数的期望**（不需要先求 Y=g(X) 的分布！）：
- E(g(X)) = Σ g(xₖ)·pₖ（离散）
- E(g(X)) = ∫ g(x)·f(x)dx（连续）

**期望的线性性质**（最重要的性质）：
1. E(aX+b) = aE(X) + b
2. E(X+Y) = E(X) + E(Y)（**永远成立，不需要独立！**）
3. 若 X,Y 独立，则 E(XY) = E(X)·E(Y)

### 方差 D(X)

**定义**：D(X) = E[(X-E(X))²]

**计算常用公式**（比定义好算！）：D(X) = E(X²) - [E(X)]²

**方差性质**：
1. D(aX+b) = a²·D(X)（常数 b 不影响方差！）
2. D(X+Y) = D(X) + D(Y) + 2Cov(X,Y)
3. 若 X,Y 独立（或不相关），则 D(X+Y) = D(X) + D(Y)

### 协方差 Cov(X,Y)

Cov(X,Y) = E[(X-E(X))(Y-E(Y))] = E(XY) - E(X)E(Y)

性质：
- Cov(X,X) = D(X)
- Cov(aX, bY) = ab·Cov(X,Y)
- Cov(X₁+X₂, Y) = Cov(X₁,Y) + Cov(X₂,Y)
- X,Y 独立 ⇒ Cov(X,Y)=0。但反过来不一定！

### 相关系数 ρ(X,Y)

ρ = Cov(X,Y) / √(D(X)·D(Y)) ∈ [-1, 1]

- ρ > 0：正相关（一个增，另一个也增）
- ρ < 0：负相关
- ρ = 0：不相关（无**线性**关系，但可能有非线性关系）
- |ρ| = 1：X 和 Y 几乎确定地有线性关系 Y=aX+b

**独立 vs 不相关**：
- 独立 ⇒ 不相关（Cov=0 ⇒ ρ=0）
- 不相关 ⇏ 独立！反例：X~N(0,1), Y=X²。ρ=0 但 Y 完全由 X 决定

### 矩

- k 阶原点矩：E(X^k)
- k 阶中心矩：E[(X-E(X))^k]
- 期望 = 一阶原点矩，方差 = 二阶中心矩
- 偏度 Skewness = E[(X-μ)³]/σ³（三阶标准化中心矩，衡量不对称性）
- 峰度 Kurtosis = E[(X-μ)⁴]/σ⁴（四阶，衡量尾部厚度）

## 规则或公式

| 公式 | 条件 |
|------|------|
| E(X+Y) = E(X)+E(Y) | 无条件成立 |
| E(aX+b) = aE(X)+b | 无条件成立 |
| D(X) = E(X²)-[E(X)]² | 无条件成立 |
| D(aX+b) = a²D(X) | 无条件成立 |
| D(X+Y) = D(X)+D(Y) | X,Y独立/不相关 |
| Cov(X,Y) = E(XY)-E(X)E(Y) | 无条件成立 |

## 完整例子

**离散型期望**：掷骰子的期望点数。
E(X) = 1×1/6 + 2×1/6 + ... + 6×1/6 = 21/6 = 3.5

**二项分布**：X~B(n,p)。利用二项分布的可加性：X = X₁+...+Xₙ，Xᵢ~B(1,p)。
E(X) = ΣE(Xᵢ) = np, D(X) = ΣD(Xᵢ) = np(1-p)

**正态分布**：X~N(μ,σ²)，直接有 E(X)=μ, D(X)=σ²。

**相关系数为零但有关联**：X~N(0,1), Y=X²。
Cov(X,Y) = E(X·X²)-E(X)·E(X²) = E(X³)-0·1 = 0（正态分布的奇数阶中心矩为0）
ρ=0，但 Y 完全由 X 决定——不相关但有关联！

## 容易出错的地方

- **E(X²) ≠ [E(X)]²**：这是最常见的错误。两者差 D(X)
- **期望的线性性不需要独立**：E(X+Y)=E(X)+E(Y) 无条件成立，这是很多推导的突破口
- **方差不是线性的**：D(aX+b)=a²D(X)，常数加 b 不改变方差（方差衡量"离散程度"，平移不改变离散程度）
- **相关系数只衡量线性相关**：ρ=0 只说明没有线性关系，不说明没有关系
