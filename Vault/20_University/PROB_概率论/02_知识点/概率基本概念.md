# 概率基本概念

## 一句话解释

概率论研究随机现象的统计规律性。概率的公理化定义（柯尔莫哥洛夫，1933年）奠定了现代概率论的数学基础：概率是定义在事件域上的、满足非负性、规范性和可列可加性的测度。

## 核心原理

### 四种概率定义

**1. 古典概型（等可能概型）**
- 条件：样本空间有限，每个样本点等可能
- P(A) = |A| / |Ω| = A 包含的样本点数 / 样本点总数
- 核心工具：排列组合。加法原理（分类）、乘法原理（分步）
- 局限：要求有限且等可能，现实中大多不满足

**2. 几何概型**
- 条件：样本空间是某个度量不为零的区域，每个点等可能
- P(A) = μ(A) / μ(Ω)，μ 可以是长度、面积、体积
- 经典问题：Buffon 投针问题、约会问题
- 注意：几何概型中概率为 0 的事件不一定是不可能事件

**3. 统计定义**
- 频率 f_n(A) = n_A / n（n 次试验中 A 出现的次数）
- 频率的稳定性：n 足够大时，f_n(A) 稳定在某个常数附近
- 这个常数就是概率 P(A)

**4. 公理化定义（柯尔莫哥洛夫）**
- 设 Ω 是样本空间，F 是事件域（σ-代数）
- 概率 P 是 F → [0,1] 的映射，满足：
  (1) 非负性：P(A) ≥ 0
  (2) 规范性：P(Ω) = 1
  (3) 可列可加性：若 A₁,A₂,... 两两互不相容，则 P(∪Aᵢ) = ΣP(Aᵢ)

### 核心公式

**加法公式**：
- 两个事件：P(A∪B) = P(A) + P(B) - P(AB)
- 三个事件：P(A∪B∪C) = P(A)+P(B)+P(C) - P(AB)-P(AC)-P(BC) + P(ABC)
- n 个事件的容斥原理（奇加偶减）

**条件概率**：
- P(A|B) = P(AB) / P(B)，其中 P(B) > 0
- 条件概率也是概率，满足概率的三条公理

**乘法公式**：
- P(AB) = P(A|B)·P(B) = P(B|A)·P(A)
- 推广：P(A₁A₂...Aₙ) = P(A₁)·P(A₂|A₁)·P(A₃|A₁A₂)·...·P(Aₙ|A₁...Aₙ₋₁)

**全概率公式**：
若 B₁,B₂,...,Bₙ 是 Ω 的一个划分（两两互斥且并集为 Ω），P(Bᵢ) > 0：
P(A) = Σᵢ P(A|Bᵢ)·P(Bᵢ)

用途：把一个复杂事件 A 的概率分解为在不同"原因"Bᵢ 下的条件概率之和。

**贝叶斯公式**（逆概率公式）：
P(Bᵢ|A) = P(A|Bᵢ)·P(Bᵢ) / Σⱼ P(A|Bⱼ)·P(Bⱼ)

贝叶斯公式的哲学意义："先验概率"P(Bᵢ) 在观察到证据 A 后，更新为"后验概率"P(Bᵢ|A)。

### 独立性

**两事件独立**：P(AB) = P(A)·P(B)
- 独立是概率关系，不是集合关系
- 独立 ≠ 互不相容（互不相容时 P(AB)=0，除非 P(A)=0 或 P(B)=0，否则不独立）

**三事件两两独立 vs 相互独立**：
- 两两独立：P(AB)=P(A)P(B), P(AC)=P(A)P(C), P(BC)=P(B)P(C)
- 相互独立：两两独立 + P(ABC)=P(A)P(B)P(C)
- 反例（Bernstein）：一个四面体，三面各涂一种颜色，第四面涂三种颜色。三事件两两独立但不相互独立

## 前置知识

- 排列组合：排列 A(n,k)=n!/(n-k)!，组合 C(n,k)=n!/[k!(n-k)!]

## 规则或公式

| 公式 | 条件 |
|------|------|
| P(A∪B) = P(A)+P(B)-P(AB) | 任意事件 |
| P(A∪B) = P(A)+P(B) | A,B 互斥 |
| P(AB) = P(A\|B)P(B) | P(B)>0 |
| P(A) = ΣP(A\|Bᵢ)P(Bᵢ) | {Bᵢ} 是划分 |
| P(B\|A) = P(A\|B)P(B)/P(A) | P(A)>0 |

## 完整例子

**猜谜问题**：三个门后有一个奖品。你选了一个门，主持人打开另一个空门。你是否应该换门？

- 设 A₁,A₂,A₃ 分别表示奖品在门1,2,3后，P(Aᵢ)=1/3
- 你选门1。设 B = 主持人打开门2
- P(B|A₁) = 1/2（奖品在门1，主持人随机开2或3）
- P(B|A₂) = 0（奖品在门2，主持人不能开）
- P(B|A₃) = 1（奖品在门3，主持人只能开门2）
- P(B) = 1/3×1/2 + 1/3×0 + 1/3×1 = 1/2
- P(A₃|B) = (1/3×1)/(1/2) = 2/3
- P(A₁|B) = (1/3×1/2)/(1/2) = 1/3
- **结论：换门的胜率 2/3，不换 1/3。应该换！**

## 容易出错的地方

- **条件概率的分母是条件的概率**：P(A|B) 不是 P(AB)，是 P(AB)/P(B)。很多人直接把 P(AB) 当条件概率
- **独立和互斥不要混淆**：互斥 = 不能同时发生，独立 = 发生概率互不影响。互斥事件如果概率都不为0，一定不独立
- **贝叶斯公式的分母是全概率**：经常有人忘了除分母，只写 P(B|A) 正比于 P(A|B)P(B)
- **条件概率的"条件"要写清楚**：P(A|B) 读作"在 B 发生的条件下 A 的概率"，两个事件角色不同
