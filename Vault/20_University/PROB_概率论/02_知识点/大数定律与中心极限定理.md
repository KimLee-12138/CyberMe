# 大数定律与中心极限定理

## 一句话解释

大数定律回答了"为什么频率会稳定在概率附近"——样本均值收敛到期望。中心极限定理解释了"为什么正态分布如此普遍"——大量独立因素的叠加近似正态分布。两者共同构成了统计推断的理论根基。

## 核心原理

### 切比雪夫不等式

P{|X - E(X)| ≥ ε} ≤ D(X) / ε²

等价形式：P{|X - E(X)| < ε} ≥ 1 - D(X)/ε²

意义：
- 方差越小，随机变量越集中在期望附近
- 这个不等式**对所有有方差的随机变量成立**，不依赖具体分布
- 但给出的界比较宽松（实际概率通常远小于上界）

### 大数定律（Law of Large Numbers）

核心思想：n 个独立同分布随机变量的算术平均值，当 n→∞ 时收敛到期望。

**(1) 辛钦大数定律（弱大数定律）**：
设 X₁,X₂,... i.i.d（独立同分布），E(Xᵢ)=μ 存在。
则 (1/n)ΣXᵢ → μ（**依概率收敛**）。

即：∀ε>0, lim_{n→∞} P{|(1/n)ΣXᵢ - μ| < ε} = 1

**(2) 伯努利大数定律**（辛钦的特例）：
n 次独立重复试验，事件 A 出现 n_A 次。
则 n_A/n → P(A)（频率收敛到概率）。

这解释了为什么概率的统计定义是合理的——长期频率确实会稳定在概率附近。

**(3) 强大数定律**（柯尔莫哥洛夫）：
P{lim_{n→∞} (1/n)ΣXᵢ = μ} = 1（**几乎必然收敛**，比依概率收敛更强）。

**大数定律的应用**：蒙特卡洛方法——用大量随机抽样来近似计算积分。∫f(x)dx ≈ (1/n)Σf(Xᵢ)。

### 中心极限定理（Central Limit Theorem）

设 X₁,X₂,...,Xₙ i.i.d，E(Xᵢ)=μ, D(Xᵢ)=σ² > 0。
令 Sₙ = X₁+...+Xₙ。

则对任意实数 x：
lim_{n→∞} P{(Sₙ - nμ) / (σ√n) ≤ x} = Φ(x) = (1/√(2π))∫_{-∞}^x e^{-t²/2} dt

**等价表述**：
- 当 n 充分大时，Sₙ ≈ N(nμ, nσ²)
- 样本均值 Ẋ ≈ N(μ, σ²/n)
- (Ẋ - μ) / (σ/√n) ≈ N(0,1)

**推论（De Moivre-Laplace）**：
若 X~B(n,p)，当 n 充分大时，X ≈ N(np, np(1-p))。
即 (X-np)/√(np(1-p)) ≈ N(0,1)。

### 三大定理的关系

```
切比雪夫不等式 ──→ 大数定律（定性：收敛到期望）
        ↓
中心极限定理（定量：近似正态分布 + 收敛速度）
```

- 大数定律告诉你会收敛
- 中心极限定理告诉你收敛后的分布和速度
- 切比雪夫不等式给出概率的粗糙上界

## 规则或公式

- 切比雪夫：P{|X-μ|≥ε} ≤ σ²/ε²
- 大数定律：(1/n)ΣXᵢ → μ（依概率）
- 中心极限：(ΣXᵢ - nμ)/(σ√n) → N(0,1)（依分布）
- 二项近似正态：X~B(n,p) ≈ N(np, np(1-p))（np≥5, n(1-p)≥5）

## 完整例子

**掷硬币 10000 次**，正面概率 0.5。求正面次数在 4900-5100 之间的概率。

- μ=5000, σ=√(10000×0.5×0.5)=50
- 中心极限：P{4900≤X≤5100} = P{-2≤(X-5000)/50≤2} ≈ Φ(2)-Φ(-2) = 2Φ(2)-1 ≈ 0.9544
- 约 95.4% 的概率正面次数在 4900-5100

**蒙特卡洛求 π**：在 [0,1]×[0,1] 中随机撒点，统计落在 1/4 单位圆内的比例。

- n 个点中 k 个落在圆内
- k/n ≈ π/4
- π ≈ 4k/n
- 大数定律保证 n→∞ 时收敛

## 容易出错的地方

- **大数定律是极限性质**：n 必须足够大。n=10 时频率可能远离概率
- **中心极限定理要求独立同分布**：不是所有分布的和都近似正态。柯西分布没有期望，其样本均值仍是柯西分布
- **正态近似的条件**：二项分布近似正态要求 np≥5 且 n(1-p)≥5
- **收敛的含义不同**：依概率收敛（弱）、几乎必然收敛（强）、依分布收敛——三种收敛不要混淆
