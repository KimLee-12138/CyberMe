# 数理统计基础

## 一句话解释

数理统计用样本数据推断总体特征。三大抽样分布（χ², t, F）是区间估计和假设检验的基础。矩估计用样本矩替代总体矩，极大似然估计选最可能产生观测数据的参数值。

## 核心原理

### 总体与样本

- **总体**：研究对象的全体。用随机变量 X 表示
- **样本**：从总体中抽取的 n 个个体 X₁,X₂,...,Xₙ。要求**独立同分布**（i.i.d）
- **统计量**：样本的函数 g(X₁,...,Xₙ)，不含未知参数。如样本均值 Ẋ、样本方差 S²

**常用统计量**：
- 样本均值：Ẋ = (1/n)ΣXᵢ
- 样本方差：S² = [1/(n-1)]Σ(Xᵢ - Ẋ)²（分母 n-1 保证无偏性！）
- 样本 k 阶矩：Aₖ = (1/n)ΣXᵢ^k

### 三大抽样分布（正态总体下）

**(1) 卡方分布 χ²(n)**

若 Z₁,...,Zₙ i.i.d ~ N(0,1)，则 χ² = ΣZᵢ² ~ χ²(n)
- E(χ²) = n, D(χ²) = 2n
- n→∞ 时，χ²(n) → N(n, 2n)（中心极限定理）
- 应用：单个正态总体方差的区间估计

**(2) t 分布 t(n)**

若 Z~N(0,1), V~χ²(n) 独立，则 t = Z/√(V/n) ~ t(n)
- t 分布关于 0 对称，尾部比 N(0,1) 厚
- n→∞ 时，t(n) → N(0,1)（n≥30 可用正态近似）
- **应用**：正态均值 μ 的区间估计（σ² 未知时）
- Ẋ ± t_{α/2}(n-1)·s/√n

**(3) F 分布 F(m, n)**

若 U~χ²(m), V~χ²(n) 独立，则 F = (U/m)/(V/n) ~ F(m, n)
- 应用：两个正态总体方差比的检验、ANOVA

### 点估计

**(1) 矩估计法**：
用样本矩代替总体矩，解出参数的估计值。
- E(X) = Ẋ → 估计 μ
- D(X) = S² → 估计 σ²

优点：直观简单。缺点：不一定是"最优"的。

**(2) 极大似然估计（MLE）**：
选使得**观测到当前样本的概率最大**的参数值。

似然函数：L(θ) = f(x₁;θ)·f(x₂;θ)·...·f(xₙ;θ)（独立样本的联合密度）
对数似然：ln L(θ) = Σ ln f(xᵢ;θ)

求 ln L(θ) 的最大值 → 令导数为零 → 解得 θ̂_{MLE}

**MLE 的性质**：
- 一致性：n→∞ 时收敛到真值
- 渐近正态性：θ̂ ≈ N(θ, 1/I(θ))（I 是 Fisher 信息量）
- 渐近有效性：在正则条件下，MLE 渐近达到 Cramér-Rao 下界

### 区间估计

给出参数 θ 的置信区间 [L, U]，使得 P{L ≤ θ ≤ U} = 1-α（置信水平）。

**正态均值 μ 的区间估计**：
- σ² 已知：Ẋ ± z_{α/2}·σ/√n
- σ² 未知：Ẋ ± t_{α/2}(n-1)·s/√n

**正态方差 σ² 的区间估计**：
[(n-1)S²/χ²_{α/2}(n-1), (n-1)S²/χ²_{1-α/2}(n-1)]

### 假设检验

**基本思想**：提出 H₀（原假设），通过样本判断是否拒绝 H₀。

**步骤**：
1. 提出 H₀ 和 H₁（备择假设）
2. 选择检验统计量和显著性水平 α（常用 0.05）
3. 计算 p 值：在原假设下，观察到当前结果或更极端结果的概率
4. 若 p < α → 拒绝 H₀；否则不拒绝

**两类错误**：
- 第 I 类（拒真）：H₀ 为真但拒绝 H₀。概率 = α（可控）
- 第 II 类（取伪）：H₀ 为假但未拒绝。概率 = β

**注意**：不拒绝 H₀ ≠ H₀ 成立。只是"现有证据不足以拒绝"。

## 规则或公式

| 统计量 | 公式 |
|--------|------|
| 样本均值 | Ẋ = ΣXᵢ/n |
| 样本方差 | S² = Σ(Xᵢ-Ẋ)²/(n-1) |
| μ 的置信区间(σ已知) | Ẋ ± z_{α/2}·σ/√n |
| μ 的置信区间(σ未知) | Ẋ ± t_{α/2}(n-1)·s/√n |

## 完整例子

**工厂生产灯泡寿命** X~N(μ,σ²)。随机抽 16 只，平均寿命 Ẋ=1500 小时，样本标准差 s=80 小时。求 μ 的 95% 置信区间。

- 1-α=0.95, α=0.05, t_{0.025}(15)=2.131
- 1500 ± 2.131×80/4 = 1500 ± 42.62
- [1457.38, 1542.62]
- 有 95% 把握认为真实平均寿命在 1457-1543 小时之间

## 容易出错的地方

- **样本方差的除数是 n-1 不是 n**：除以 n 是总体方差（有偏估计），除以 n-1 才是无偏估计
- **p 值不是 H₀ 为真的概率**：p 值是"假定 H₀ 为真，观察到该结果或更极端的概率"
- **不拒绝 ≠ 接受**：不能证明虚无假设。只能说"证据不足"
- **t 分布和正态分布的选择**：σ² 已知用 z，未知用 t。大样本（n≥30）t≈z
