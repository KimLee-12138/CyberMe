# 范式判断与候选码求解 — 例题集

## 例题1：候选码求解

**来源**：AI 生成练习（`problem_origin: ai-generated`）

**题目**：R(A, B, C, D)，F = {A→B, B→C, C→D}，求所有候选码。

**解题步骤**：
1. 属性分类：L类={A}，R类={D}，LR类={B, C}，N类=∅
2. 计算 L∪N 的闭包：A⁺ = {A, B, C, D} = U
3. A⁺ = U → {A} 是唯一候选码

**答案**：候选码为 {A}

**易错点**：不要漏掉闭包计算步骤，不能仅凭看就判断候选码。

## 例题2：判断范式等级

**来源**：AI 生成练习（`problem_origin: ai-generated`）

**题目**：R(sno, cno, sname, sdept, grade)，F = {sno→sname, sno→sdept, (sno,cno)→grade}，判断R属于第几范式。

**解题步骤**：
1. 找候选码：{sno, cno}⁺ = {sno, cno, sname, sdept, grade} = U，所以候选码为 {sno, cno}
2. 非主属性：sname, sdept, grade
3. 检查部分依赖：sno→sname，sno是候选码的真子集 → 存在部分依赖
4. 结论：R ∈ 1NF（不满足2NF）

**答案**：R 属于 1NF

**易错点**：候选码是 {sno,cno} 而不是 sno——grade 需要同时知道 sno 和 cno 才能确定。

## 例题3：BCNF判断

**来源**：AI 生成练习（`problem_origin: ai-generated`）

**题目**：R(sno, cno, tno)，假设每位老师只教一门课。F = {(sno,cno)→tno, tno→cno}。判断R是否属于BCNF。

**解题步骤**：
1. 找候选码：{sno, cno}⁺ = U，{sno, tno}⁺ = {sno, tno, cno} = U
   → 候选码为 {sno, cno} 和 {sno, tno}
2. 主属性：sno, cno, tno（全部都是主属性）
3. 3NF？非主属性集为空 → 自动满足3NF
4. BCNF？检查 tno→cno：
   - 决定因素 tno 是否包含候选码？{tno} 不包含 {sno,cno}，也不包含 {sno,tno} → **不是超码**
   - 违反 BCNF 条件

**答案**：R 属于 3NF，但不属于 BCNF

**易错点**：3NF 可以通过（没有非主属性），但 BCNF 要看全部FD的决定因素。这是典型"3NF但不BCNF"的案例。

## 例题4：2NF分解

**来源**：AI 生成练习（`problem_origin: ai-generated`）

**题目**：R(sno, cno, sname, grade)，F = {sno→sname, (sno,cno)→grade}。分解为2NF。

**解题步骤**：
1. 候选码：{sno, cno}（验证：闭包={sno,cno,sname,grade}=U）
2. 非主属性：sname, grade
3. sno→sname 是部分依赖（sno是候选码的真子集）→ 不满足2NF
4. 分解：
   - R1(sno, sname)，候选码 {sno}（sname完全依赖sno）
   - R2(sno, cno, grade)，候选码 {sno, cno}（grade完全依赖整个候选码）

**答案**：分解为 R1(sno, sname) 和 R2(sno, cno, grade)

**易错点**：不能把 grade 也放到 R1 中——grade 需要 cno 才能确定。

## 例题5：3NF分解（传递依赖）

**来源**：AI 生成练习（`problem_origin: ai-generated`）

**题目**：R(sno, sdept, dean)，F = {sno→sdept, sdept→dean}。判断范式并分解。

**解题步骤**：
1. 候选码：sno⁺ = {sno, sdept, dean} = U → 候选码为 {sno}
2. 完全依赖？sno→sdept 完全，sno→dean 完全？不——sno→sdept→dean，是传递依赖
3. 2NF？单属性候选码，自动满足2NF
4. 3NF？存在传递依赖 sno→sdept→dean → 不满足3NF
5. 分解：R1(sno, sdept)，R2(sdept, dean)

**答案**：R ∈ 2NF。分解为 R1(sno, sdept) 和 R2(sdept, dean)

**易错点**：sno→dean 看起来像是完全依赖（sno是单属性），但实际是传递的（通过sdept中转）。
