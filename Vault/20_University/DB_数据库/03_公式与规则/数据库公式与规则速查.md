# 数据库公式与规则速查卡

> 考前快速过一遍。每条规则已链接到详细解释。

## 关系代数

| 运算 | 符号 | 含义 |
|------|------|------|
| 选择 | σ_{条件}(R) | 横向取行 |
| 投影 | Π_{列}(R) | 纵向取列 |
| 连接 | R ⋈_{条件} S | 笛卡儿积+选择 |
| 自然连接 | R ⋈ S | 公共属性等值连接+去重 |
| 并 | R ∪ S | 同目关系的行合并 |
| 差 | R - S | 在R中不在S中 |
| 交 | R ∩ S | 同时属于R和S |

## Armstrong 公理

| 公理 | 表达式 | 含义 |
|------|--------|------|
| 自反律 | 若 Y⊆X⊆U，则 X→Y | 自己决定自己的子集 |
| 增广律 | 若 X→Y，则 XZ→YZ | 两边加相同属性 |
| 传递律 | 若 X→Y, Y→Z，则 X→Z | 依赖链传递 |

## 函数依赖类型速判

| 类型 | 条件 | 记号 |
|------|------|------|
| 平凡FD | Y ⊆ X | X→Y 平凡 |
| 完全FD | X→Y 且对 X 的任意真子集 X' 有 X'↛Y | X —F→ Y |
| 部分FD | 存在 X'⊂X 使得 X'→Y | X —P→ Y |
| 传递FD | X→Y, Y→Z, Y↛X, Z⊈Y | X —传递→ Z |

## 范式判定

| 范式 | 条件 | 考点 |
|------|------|------|
| 1NF | 属性不可再分 | 基本要求 |
| 2NF | 1NF + 非主属性完全依赖每个候选码 | 消除部分依赖 |
| 3NF | 2NF + 非主属性不传递依赖候选码 | 消除传递依赖 |
| BCNF | 每个非平凡FD的决定因素都是超码 | 消除主属性的部分/传递依赖 |

## SQL 执行顺序

```text
FROM → WHERE → GROUP BY → HAVING → SELECT → ORDER BY
  │       │         │          │         │         │
 表     行过滤    分组       组过滤    选列      排序
```

## 封锁协议层级

| 级别 | 加锁规则 | 解决的问题 |
|------|---------|-----------|
| 一级 | 写前加X锁，事务结束释放 | 丢失修改 |
| 二级 | 一级 + 读前加S锁，读完释放S锁 | +读脏数据 |
| 三级 | 一级 + 读前加S锁，事务结束释放 | +不可重复读 |

## WAL 原则

> Write-Ahead Logging：修改数据前，日志记录必须先写入稳定存储。

## 候选码求解步骤

1. 属性分四类：L（仅左）、R（仅右）、LR（双出）、N（不出）
2. L∪N 必属于任何候选码
3. 计算 (L∪N)⁺，若=U则为唯一候选码
4. 否则逐个添加LR类属性，直到闭包覆盖U
