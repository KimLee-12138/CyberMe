练习题参考答案

1.仓库管理 (1) 属于 2NF。理由：候选码为仓库号，非主属性完全依赖于码，但存在传递依赖：仓库号 -> 管理员ID -> 管理员姓名。 (2) 分解至 BCNF：消除传递依赖。结果为：R1(管理员ID, 管理员姓名)，R2(仓库号, 所在地, 面积, 管理员ID)。

2.教学 (1) 属于 1NF。理由：码为(学号, 课程号)，存在部分依赖：课程号 -> 教师名。 (2) 分解至 BCNF： 第一步(1NF -> 2NF)：消除部分依赖。分解为 R1(课程号, 教师名, 教师职称)，R2(学号, 课程号, 成绩)。 第二步(2NF -> BCNF)：R1中存在传递依赖 课程号 -> 教师名 -> 教师职称。 最终结果：R11(教师名, 教师职称)，R12(课程号, 教师名)，R2(学号, 课程号, 成绩)。

3.订单明细 (1) 属于 1NF。理由：存在部分依赖 产品号 -> 产品名。 (2) 分解至 BCNF：消除部分依赖。结果为：R1(产品号, 产品名, 单价, 供应商)，R2(订单号, 产品号, 数量)。

4.公司项目 (1) 属于 1NF。理由：存在部分依赖 项目编号 -> 项目名，以及 员工号 -> 员工姓名。 (2) 分解至 BCNF：消除所有部分依赖。结果为：R1(项目编号, 项目名)，R2(员工号, 员工姓名)，R3(项目编号, 员工号, 投入工时)。

5.城市商店 (1) 属于 2NF。理由：存在传递依赖 (商店名, 城市) -> 经理 -> 经理入职年份。 (2) 分解至 BCNF：消除传递依赖。结果为：R1(经理, 经理入职年份)，R2(商店名, 城市, 经理)。

6.抽象 R(A, B, C, D, E) (1) 属于 1NF。理由：码为(A, B)，存在部分依赖 B -> D。 (2) 分解至 BCNF：消除部分依赖。结果为：R1(B, D)，R2(A, B, C, E)。

7.抽象 R(A, B, C, D) (1) 属于 2NF。理由：存在传递依赖 (A, C) -> B -> D。 (2) 分解至 BCNF：消除传递依赖。结果为：R1(B, D)，R2(A, C, B)。

8.抽象 R(A, B, C, D) (1) 属于 1NF。理由：存在部分依赖 A -> C 和 B -> D。 (2) 分解至 BCNF：消除部分依赖。结果为：R1(A, C)，R2(B, D)，R3(A, B)。

9.抽象 R(X, Y, Z, W) (1) 不属于 3NF，它是 2NF。理由：存在非主属性 W 传递依赖于码 (XY -> Z -> W)。 (2) 不属于 BCNF。理由：存在 Z -> X，Z 不包含码。 (3) 分解结果：R1(Z, W)，R2(Z, X)，R3(Z, Y)。

10.抽象 R(A, B, C) (1) 候选码：(A, B) 和 (A, C)。 (2) 最高属于 3NF。理由：没有非主属性，所有属性都是主属性，消除了非主属性对码的部分和传递依赖。 (3) 分解至 BCNF：因为 C -> B 中 C 不是候选码。结果为：R1(C, B)，R2(A, C)。
