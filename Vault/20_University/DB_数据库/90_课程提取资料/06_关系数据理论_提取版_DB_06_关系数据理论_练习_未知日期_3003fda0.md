# DB_06_关系数据理论_练习_未知日期_3003fda0：提取版

> 本文件是检索中间材料，不是已核对的正式知识结论。

数据库规范化理论进阶练习题（需自行推导候选码）

1.设有关系 R(A, B, C, D)，函数依赖集 F = { A -> B, B -> C, A -> D }。

要求：

（1）求出 R 的候选码。

（2）分析 R 属于第几范式并说明理由。

（3）将其分解为 BCNF。

2.设有关系 R(A, B, C, D, E)，函数依赖集 F = { (A, B) -> C, C -> D, D -> E }。

要求：

（1）求出 R 的候选码。

（2）分析 R 属于第几范式？是否存在传递依赖？

（3）将其分解为 BCNF。

3.设有关系 R(A, B, C, D)，函数依赖集 F = { B -> D, (A, B) -> C }。

要求：

（1）求出 R 的候选码。

（2）分析 R 属于第几范式？是否存在部分依赖？

（3）将其分解为 BCNF。

4.设有关系 R(A, B, C, D)，函数依赖集 F = { A -> B, B -> A, A -> C, D -> A }。

要求：

（1）求出 R 的所有候选码。

（2）分析 R 最高满足第几范式？

（3）将其分解为 BCNF。

5.设有关系 R(A, B, C, D, E)，函数依赖集 F = { A -> BC, CD -> E, B -> D }。

要求：

（1）求出 R 的候选码。

（2）说明该关系是否存在部分依赖或传递依赖。

（3）分解至 BCNF。

6.设有关系 R(学号, 身份证号, 姓名, 班级号, 班长学号)。 函数依赖集 F = { 学号 -> 姓名, 学号 -> 班级号, 身份证号 -> 学号, 班级号 -> 班长学号 }。

要求：

（1）求出该关系的所有候选码。

（2）分析该关系属于第几范式。

（3）分解至 BCNF。

7.设有关系 R(A, B, C, D)，函数依赖集 F = { A -> B, B -> C, C -> D, D -> A }。

要求：

（1）求出 R 的所有候选码。

（2）说明该关系最高属于第几范式？

（3）是否需要进一步分解至 BCNF？

8.设有关系 R(A, B, C, D, E)，函数依赖集 F = { (A, B) -> C, C -> A, (A, B) -> D, D -> E }。

要求：

（1）求出 R 的候选码。

（2）分析 R 属于第几范式。

（3）分解至 BCNF。

9.设有关系 R(A, B, C, D)，函数依赖集 F = { A -> C, C -> A, B -> D }。

要求：

（1）求出 R 的候选码。

（2）分析该关系属于第几范式。

（3）分解至 BCNF。

10.设有关系 R(项目号, 零件号, 供应商号, 数量)。 已知：每个项目可以使用多个零件，每个零件可由不同供应商提供；但对于同一个项目，某种零件只能由一个指定的供应商提供。

要求：

（1）根据描述写出函数依赖集。

（2）求出候选码。

（3）分析范式等级并分解至 BCNF。

进阶练习题参考答案

1.答案： (1) 候选码：A。 (2) 属于 2NF。理由：非主属性完全依赖于码 A，但存在传递依赖 A -> B -> C。 (3) BCNF 分解：R1(A, B, D)，R2(B, C)。

2.答案： (1) 候选码：(A, B)。 (2) 属于 2NF。理由：存在传递依赖 (A, B) -> C -> D 以及 C -> D -> E。 (3) BCNF 分解：R1(A, B, C)，R2(C, D)，R3(D, E)。

3.答案： (1) 候选码：(A, B)。 (2) 属于 1NF。理由：存在部分依赖 B -> D（D 只依赖于码的一部分 B）。 (3) BCNF 分解：R1(B, D)，R2(A, B, C)。

4.答案： (1) 候选码：D。解析：D -> A，A -> B，A -> C，所以 D 可以推导出所有属性。 (2) 属于 2NF。理由：非主属性完全依赖于码 D，但存在传递依赖 D -> A -> B 和 D -> A -> C。 (3) BCNF 分解：R1(D, A)，R2(A, B, C)。

5.答案： (1) 候选码：A。解析：A -> BC, B -> D, (A, C) -> (B, C) -> (CD) -> E。 (2) 属于 2NF。理由：存在传递依赖 A -> B -> D 和 A -> (BC/CD) -> E。 (3) BCNF 分解：R1(A, B, C)，R2(B, D)，R3(C, D, E)。

6.答案： (1) 候选码：学号，身份证号。 (2) 属于 2NF。理由：非主属性完全依赖于码，但存在传递依赖 学号 -> 班级号 -> 班长学号。 (3) BCNF 分解：R1(身份证号, 学号)，R2(学号, 姓名, 班级号)，R3(班级号, 班长学号)。

7.答案： (1) 候选码：A, B, C, D（任一属性都是候选码）。 (2) 最高属于 BCNF。理由：每个函数依赖的左部都包含了候选码，且没有非主属性。 (3) 不需要分解。

8.答案： (1) 候选码：(A, B) 和 (C, B)。解析：(A, B) -> C 且 C -> A。 (2) 属于 3NF。理由：没有非主属性对码的部分或传递依赖。但不满足 BCNF，因为 C -> A 中 C 不是码。 (3) BCNF 分解：R1(C, A)，R2(C, B, D)，R3(D, E)。

9.答案： (1) 候选码：(A, B) 和 (C, B)。 (2) 属于 1NF。理由：存在部分依赖 B -> D。 (3) BCNF 分解：R1(B, D)，R2(A, B, C) [或 R2(C, B, A)]。

10.答案： (1) 函数依赖：(项目号, 零件号) -> 供应商号，(项目号, 零件号) -> 数量。 (2) 候选码：(项目号, 零件号)。 (3) 属于 BCNF。理由：所有非主属性都完全依赖于码，且左部即为候选码。
