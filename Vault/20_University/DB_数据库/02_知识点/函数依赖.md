# 函数依赖

## 一句话解释

函数依赖描述了一个属性（组）的值唯一确定另一个属性（组）的值。

## 核心原理

**函数依赖（Functional Dependency, FD）**：设 R(U) 是属性集 U 上的关系模式，X, Y ⊆ U。若对 R(U) 的任意可能关系 r，r 中不存在两个元组在 X 上值相同而在 Y 上值不同，则称 X 函数确定 Y，或 Y 函数依赖于 X，记为 X → Y。

**完全函数依赖**：X → Y，且对 X 的任何真子集 X'，X' → Y 不成立。记为 X —F→ Y。

**部分函数依赖**：X → Y，且存在 X 的真子集 X'，使 X' → Y 成立。

**传递函数依赖**：若 X → Y（Y ⊈ X），Y → Z（Z ⊈ Y），且 Y → X 不成立，则 Z 传递函数依赖于 X。

## 规则或公式

Armstrong 公理系统：
1. 自反律：若 Y ⊆ X ⊆ U，则 X → Y
2. 增广律：若 X → Y，Z ⊆ U，则 XZ → YZ
3. 传递律：若 X → Y，Y → Z，则 X → Z

## 完整例子

Student(sno, sname, sdept, dean)：
- sno → sname（学号确定姓名）→ 完全函数依赖
- (sno, cno) → grade（学号+课程号确定成绩）→ 完全函数依赖
- (sno, cno) → sname 是部分函数依赖（sno → sname 即可）
- sno → sdept, sdept → dean → sno —传递→ dean → 传递函数依赖

## 容易出错的地方

- 函数依赖是语义概念，不能仅凭当前数据推断
- Y → X 不成立是传递函数依赖的必要条件
- 平凡函数依赖（Y ⊆ X）是函数依赖的特例
