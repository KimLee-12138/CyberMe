# 数据库管理系统

## 一句话解释

DBMS 是位于用户与操作系统之间的一层数据管理软件，负责数据库的建立、使用和维护。

## 核心原理

数据库管理系统（Database Management System, DBMS）是数据库系统的核心。它提供：
- **数据定义功能**：定义数据库对象（表、视图、索引等）
- **数据操纵功能**：增删改查
- **数据库运行管理**：并发控制、安全性检查、完整性约束、故障恢复
- **数据库建立与维护**：数据加载、备份、重组、性能监控

## 前置知识

- [[数据与数据库]]

## 规则或公式

DBMS 的主要功能 → 数据定义 + 数据组织存储管理 + 数据操纵 + 数据库事务管理与运行管理 + 数据库建立维护

## 完整例子

常见 DBMS：MySQL、PostgreSQL、Oracle、SQL Server。所有 DBMS 都使用 SQL 作为标准查询语言。

## 容易出错的地方

- 把 DBMS 等同于数据库：DBMS 是软件系统，数据库是数据的集合
- 认为安装了 MySQL 就等于有了数据库：还需要创建具体的数据库实例
