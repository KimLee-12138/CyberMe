## 一、选择题（共 10 题）

### 1. 在 FPGA/CPLD 设计流程中，“综合”之后通常生成的是（ ）。
A. 源程序文件  
B. 电路网表文件  
C. 仿真波形文件  
D. 下载配置文件

**答案：B**

**卷面答案：** 综合是将 HDL 文件或原理图描述转换为与硬件结构相对应的底层电路网表文件。

---

### 2. 下列器件中，属于边沿触发、可在时钟有效边沿保存一位二进制信息的是（ ）。
A. 与门  
B. 译码器  
C. 触发器  
D. 数据选择器

**答案：C**

**卷面答案：** 触发器是时序逻辑电路中的基本存储单元，通常受时钟边沿控制，可保存一位二进制信息。

---

### 3. 在触发器符号中，时钟端有小三角形表示（ ）。
A. 电平触发  
B. 边沿触发  
C. 异步复位  
D. 三态输出

**答案：B**

**卷面答案：** 小三角形表示边沿触发；若小三角形前还有小圆圈，则表示下降沿触发；无小圆圈一般表示上升沿触发。

---

### 4. 下列关于 FPGA 的说法正确的是（ ）。
A. FPGA 只能串行执行程序  
B. FPGA 具有可编程、可重构、并行处理能力强的特点  
C. FPGA 与 ASIC 一样设计完成后不能修改  
D. FPGA 不适合复杂时序逻辑设计

**答案：B**

**卷面答案：** FPGA 具有可编程、可重构、并行处理能力强、适合复杂时序逻辑和数字系统设计等特点。

---

### 5. 下列关于同步置零和异步置零的说法正确的是（ ）。
A. 同步置零不需要等待时钟信号  
B. 异步置零必须等待时钟有效边沿  
C. 同步置零在时钟有效边沿到来时起作用  
D. 同步置零和异步置零完全相同

**答案：C**

**卷面答案：** 同步置零信号需要等待时钟有效边沿才起作用；异步置零信号不必等待时钟有效边沿，只要复位信号有效即可立即置零。

---

### 6. EDA 数字系统设计中，自上而下的设计方法是指（ ）。
A. 先画门电路，再写系统功能  
B. 从系统整体功能出发，逐层分解为子模块  
C. 只使用原理图设计  
D. 只使用汇编语言设计

**答案：B**

**卷面答案：** 自上而下设计方法是将数字系统整体分解为子系统和模块，逐层细化，直到便于逻辑电路设计和实现。

---

### 7. 关于 8421 码，下列说法正确的是（ ）。
A. 8421 码是普通纯二进制码  
B. 8421 码是最常用的 BCD 码，每 4 位表示 1 位十进制数  
C. 8421 码可以直接表示任意十六进制数  
D. 8421 码没有非法状态

**答案：B**

**卷面答案：** 8421 码是 BCD 码的一种，4 位二进制权值分别为 8、4、2、1，用来表示一位十进制数 0~9，1010~1111 为非法状态。

---

### 8. 下列 Verilog 语句中，属于连续赋值、常用于数据流描述的是（ ）。
A. always  
B. initial  
C. assign  
D. case

**答案：C**

**卷面答案：** assign 语句用于连续赋值，常用于数据流描述；always 语句常用于行为描述。

---

### 9. Moore 型状态机的输出取决于（ ）。
A. 只取决于当前状态  
B. 只取决于输入  
C. 取决于当前状态和输入  
D. 与状态和输入均无关

**答案：A**

**卷面答案：** Moore 型状态机的输出只与当前状态有关；Mealy 型状态机的输出与当前状态和输入均有关。

---

### 10. 若用缩位异或运算 `^a` 对 8 位信号 `a` 进行运算，其常见用途是（ ）。
A. 判断奇偶校验  
B. 完成乘法  
C. 产生三态输出  
D. 进行移位寄存

**答案：A**

**卷面答案：** 缩位异或可将向量各位依次异或，常用于判断数据中 1 的个数奇偶性，从而产生奇偶校验位。

## 二、大题一：组合逻辑电路分析与化简

### 【恢复题干】
已知如下 Verilog 结构描述程序：

```verilog
module sy(A,B,C,D,F);
input A,B,C,D;
output F;
wire F1,F2;
and  a1(F2,B,C,D);
nand n1(F1,A,B);
or   o1(F,F1,F2);
endmodule
```

要求：

1. 根据 Verilog 代码写出输出逻辑表达式；
2. 用公式法化简，并说明卡诺图化简结果；
3. 根据最简表达式画出逻辑电路图。

### 【参考答案】

#### 1. 写出逻辑表达式

由程序可知：

$$
F_2=BCD
$$

$$
F_1=(AB)'
$$

$$
F=F_1+F_2=(AB)'+BCD
$$

因此输出逻辑表达式为：

$$
F=(AB)'+BCD
$$

#### 2. 公式法化简

$$
F=(AB)'+BCD
$$

由德摩根定律：

$$
(AB)'=A'+B'
$$

所以：

$$
F=A'+B'+BCD
$$

利用公式 $X+X'Y=X+Y$，令 $X=B'$，则：

$$
B'+BCD=B'+CD
$$

因此：

$$
F=A'+B'+CD
$$

**最简表达式：**

$$
\boxed{F=A'+B'+CD}
$$

卡诺图化简结果与公式化简结果一致，均为：

$$
\boxed{F=A'+B'+CD}
$$

#### 3. 电路图文字说明

根据最简式：

$$
F=A'+B'+CD
$$

画图时：

1. A 经过非门得到 $A'$；
2. B 经过非门得到 $B'$；
3. C、D 经过二输入与门得到 $CD$；
4. 将 $A'$、$B'$、$CD$ 三项输入三输入或门，输出即为 F。

## 三、大题二：序列产生器设计

### 【恢复题干】
设计一个序列产生器，使输出端 `z` 在时钟作用下循环输出序列 `1001`。要求画出状态转移关系，并写出 Verilog 程序。

> 注：回忆版只记为“序列产生器”，没有保留具体序列。本题按最常见的循环序列产生器题型恢复，序列取 `1001`。若原题给出其他序列，只需替换状态输出即可。

### 【参考答案】

#### 1. 状态定义

设计 4 个状态：

- `S0`：输出 1；
- `S1`：输出 0；
- `S2`：输出 0；
- `S3`：输出 1。

状态转移为：

$$
S0 \rightarrow S1 \rightarrow S2 \rightarrow S3 \rightarrow S0
$$

输出序列为：

$$
1,0,0,1,1,0,0,1,\cdots
$$

#### 2. Verilog 程序

```verilog
module seq_gen_1001(
    input  wire clk,
    input  wire rst,
    output reg  z
);
    reg [1:0] cs, ns;
    parameter S0 = 2'd0,
              S1 = 2'd1,
              S2 = 2'd2,
              S3 = 2'd3;

    // 状态寄存器
    always @(posedge clk or posedge rst) begin
        if (rst)
            cs <= S0;
        else
            cs <= ns;
    end

    // 下一状态逻辑
    always @(*) begin
        case (cs)
            S0: ns = S1;
            S1: ns = S2;
            S2: ns = S3;
            S3: ns = S0;
            default: ns = S0;
        endcase
    end

    // 输出逻辑
    always @(*) begin
        case (cs)
            S0: z = 1'b1;
            S1: z = 1'b0;
            S2: z = 1'b0;
            S3: z = 1'b1;
            default: z = 1'b0;
        endcase
    end
endmodule
```

#### 3. 卷面总结

本题采用 Moore 型状态机，输出只与当前状态有关。四个状态依次循环，分别输出 1、0、0、1，因此可循环产生 `1001` 序列。

## 四、大题三：材料型组合逻辑电路设计

### 【恢复题干】
阅读题目给出的功能描述材料，完成组合逻辑电路设计。可能涉及编码器、译码器、数据选择器、加法器、比较器或奇偶校验电路。要求写出输入输出变量、逻辑表达式，并给出 Verilog 程序。

> 注：回忆版只写“组合逻辑电路，给一段材料分析”，没有保留原材料。根据复习资料，本题最可能考“数据选择器/比较器/编码器/奇偶校验/加法器”等组合逻辑设计。下面给出最通用的卷面作答模板。

### 【参考答案模板一：4 选 1 数据选择器】

#### 1. 功能分析

设选择信号为 `sel[1:0]`，数据输入为 `d0、d1、d2、d3`，输出为 `y`。

当：

- `sel=00` 时，`y=d0`；
- `sel=01` 时，`y=d1`；
- `sel=10` 时，`y=d2`；
- `sel=11` 时，`y=d3`。

#### 2. 逻辑表达式

$$
y=d_0\overline{sel_1}\overline{sel_0}+d_1\overline{sel_1}sel_0+d_2sel_1\overline{sel_0}+d_3sel_1sel_0
$$

#### 3. Verilog 程序

```verilog
module mux4_1(
    input  wire [1:0] sel,
    input  wire d0, d1, d2, d3,
    output reg  y
);
    always @(*) begin
        case (sel)
            2'b00: y = d0;
            2'b01: y = d1;
            2'b10: y = d2;
            2'b11: y = d3;
            default: y = 1'b0;
        endcase
    end
endmodule
```

### 【参考答案模板二：4 位比较器】

```verilog
module compare4(
    input  wire [3:0] A,
    input  wire [3:0] B,
    output wire [2:0] outcome
);
    // outcome[0]=1 表示 A>B
    // outcome[1]=1 表示 A=B
    // outcome[2]=1 表示 A<B
    assign outcome[0] = (A > B);
    assign outcome[1] = (A == B);
    assign outcome[2] = (A < B);
endmodule
```

**卷面总结：** 组合逻辑电路的输出只取决于当前输入，不含存储功能。设计步骤为：分析功能、确定输入输出变量、列真值表、化简逻辑表达式、画电路图或写 Verilog 程序。

## 五、大题四：1001 序列检测器设计

### 【恢复题干】
设计一个串行序列检测器。输入为 `x`，输出为 `z`。当输入串中检测到连续序列 `1001` 时，输出 `z=1`，否则 `z=0`。要求画出状态转移图，并写出 Verilog 程序。

本题按 Mealy 型状态机设计，允许序列重叠检测。

### 【参考答案】

#### 1. 状态定义

设状态含义如下：

- `S0`：未检测到有效前缀；
- `S1`：已检测到 `1`；
- `S2`：已检测到 `10`；
- `S3`：已检测到 `100`。

当处于 `S3` 且输入 `x=1` 时，说明检测到 `1001`，输出 `z=1`。

#### 2. 状态转移关系

| 当前状态 | 输入 x=0 | 输入 x=1 |
|---|---|---|
| S0 | S0 / 0 | S1 / 0 |
| S1 | S2 / 0 | S1 / 0 |
| S2 | S3 / 0 | S1 / 0 |
| S3 | S0 / 0 | S1 / 1 |

其中表中“次态 / 输出”表示在该输入下转移到的状态和输出值。

#### 3. Verilog 程序

```verilog
module seq_det_1001(
    input  wire clk,
    input  wire rst,
    input  wire x,
    output reg  z
);
    reg [1:0] cs, ns;
    parameter S0 = 2'd0,
              S1 = 2'd1,
              S2 = 2'd2,
              S3 = 2'd3;

    // 状态寄存器
    always @(posedge clk or posedge rst) begin
        if (rst)
            cs <= S0;
        else
            cs <= ns;
    end

    // 下一状态逻辑
    always @(*) begin
        case (cs)
            S0: begin
                if (x) ns = S1;
                else   ns = S0;
            end
            S1: begin
                if (x) ns = S1;
                else   ns = S2;
            end
            S2: begin
                if (x) ns = S1;
                else   ns = S3;
            end
            S3: begin
                if (x) ns = S1;
                else   ns = S0;
            end
            default: ns = S0;
        endcase
    end

    // 输出逻辑：Mealy 型，输出与当前状态和输入有关
    always @(*) begin
        if (cs == S3 && x == 1'b1)
            z = 1'b1;
        else
            z = 1'b0;
    end
endmodule
```

#### 4. 卷面总结

该电路为 Mealy 型序列检测器，输出与当前状态和输入有关。状态分别表示已经匹配的最长有效前缀，当状态为 `S3` 且输入为 `1` 时，输入序列刚好形成 `1001`，因此输出 `z=1`。
