# 代数系统

> 一句话：群论是研究运算结构的数学分支，布尔代数是数字逻辑电路的理论基础。

## 一、代数系统基础

**代数系统**：集合 + 其上的运算。

### 1.1 运算的性质

| 性质 | 定义 |
|------|------|
| **封闭** | $\forall a,b\in S, a\circ b\in S$ |
| **结合律** | $(a\circ b)\circ c = a\circ(b\circ c)$ |
| **交换律** | $a\circ b = b\circ a$ |
| **单位元 e** | $\forall a, e\circ a = a\circ e = a$ |
| **逆元** | $\forall a, \exists a^{-1}: a\circ a^{-1} = a^{-1}\circ a = e$ |
| **幂等律** | $a\circ a = a$ |
| **分配律** | $a\circ(b*c) = (a\circ b)*(a\circ c)$ |

## 二、半群与独异点

| 结构 | 条件 |
|------|------|
| **半群** | 封闭 + 结合律 |
| **独异点(含幺半群)** | 半群 + 有单位元 |

## 三、群

### 3.1 定义
封闭 + 结合律 + 单位元 + **每个元素有逆元**。

### 3.2 群的性质

- 单位元唯一
- 每个元素的逆元唯一
- $(ab)^{-1} = b^{-1}a^{-1}$
- 满足消去律：$ab=ac \Rightarrow b=c$

### 3.3 子群

$H \subseteq G$ 是子群 $\Leftrightarrow$：
1. $H$ 非空
2. $\forall a,b\in H: ab^{-1} \in H$（一个条件判别法）

### 3.4 特殊群

| 类型 | 额外条件 |
|------|---------|
| **交换群(Abel群)** | 满足交换律 |
| **循环群** | $\exists a\in G: G = \{a^n \mid n\in Z\}$ |

### 3.5 拉格朗日定理

若 $H$ 是有限群 $G$ 的子群，则 $|H|$ 整除 $|G|$。

**推论**：$|a|$（元素 a 的阶）整除 $|G|$（群的阶）。

## 四、环与域

### 4.1 环
两个二元运算：加法(构成交换群) + 乘法(构成半群)，满足分配律。

### 4.2 域
环 + 乘法构成交换群(非零元)。

例：实数域、有理数域。

## 五、布尔代数

### 5.1 定义
满足交换律、分配律、同一律、补充律的代数系统。

两个二元运算：$\land$（与）、$\lor$（或）；一元运算：$'$（非）。

### 5.2 基本定律

| 定律 | 公式 |
|------|------|
| 交换律 | $a\land b = b\land a$ |
| 分配律 | $a\land(b\lor c) = (a\land b)\lor(a\land c)$ |
| 同一律 | $a\land 1 = a, a\lor 0 = a$ |
| 补充律 | $a\land a' = 0, a\lor a' = 1$ |
| **德摩根律** | $(a\land b)' = a'\lor b'$ |

### 5.3 布尔代数与数字逻辑

- **命题逻辑**是布尔代数的特例
- **数字电路**：1→高电平，0→低电平，$\land$→与门，$\lor$→或门，$'$→非门
