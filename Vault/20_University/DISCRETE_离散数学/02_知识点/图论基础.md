# 图论基础

> 一句话：图是由顶点和边组成的抽象结构，是建模网络、路径、匹配等问题的核心工具。

## 一、图的基本概念

### 1.1 定义
**无向图** $G = (V, E)$，$V$ 为顶点集，$E$ 为边集(无序对)。

**有向图** $D = (V, E)$，$E$ 为有向边集(有序对)。

### 1.2 基本术语

| 术语 | 定义 |
|------|------|
| **度 d(v)** | 与顶点 v 关联的边数 |
| **握手定理** | $\sum_{v\in V} d(v) = 2|E|$（度之和 = 2倍边数） |
| **出/入度** | 有向图中分出/进入的边数 |
| **简单图** | 无自环、无平行边 |
| **完全图 $K_n$** | 每对顶点间都有边 |
| **补图** | $\bar{G}$：边存在的变没有，不存在的变有 |

### 1.3 握手定理推论

**任何图中奇度顶点的个数是偶数**。这一定理是图论中最简单但最有用的结论之一。

## 二、路径与连通

### 2.1 基本概念

| 术语 | 定义 |
|------|------|
| **通路(Path)** | 顶点序列 $v_0,e_1,v_1,...,e_n,v_n$，边不重复 |
| **简单通路** | 顶点不重复 |
| **回路(Cycle)** | 起点=终点的通路 |
| **连通图** | 任意两顶点间存在通路 |

### 2.2 连通性
- **连通分量**：极大连通子图
- **割点**：删除后连通分量增加的顶点
- **桥(割边)**：删除后连通分量增加的边

## 三、图的矩阵表示

### 3.1 邻接矩阵

$A[i][j] = $ 从 $v_i$ 到 $v_j$ 的边数。

**性质**：$A^k[i][j]$ = 从 $v_i$ 到 $v_j$ 长度为 $k$ 的通路数。

### 3.2 关联矩阵

$M[i][j] = $ 1（若 $v_i$ 关联边 $e_j$），0（否则）。

## 四、欧拉图与哈密顿图

### 4.1 欧拉通路/回路
经过每条边恰好一次。

| 类型 | 无向图条件 | 有向图条件 |
|------|----------|----------|
| 欧拉回路 | 所有顶点度数为**偶** | 入度=出度 |
| 欧拉通路 | 恰有0或2个奇度顶点 | 恰有一顶点入度-出度=1，一顶点出度-入度=1 |

**一笔画问题**：能用一笔画出的连通图 ↔ 有欧拉通路。

### 4.2 哈密顿通路/回路
经过每个顶点恰好一次。

**哈密顿图的充分条件（狄拉克定理）**：若 $G$ 是 $n\ge 3$ 的简单图，且 $\forall v, d(v) \ge n/2$，则 $G$ 是哈密顿图。

**奥勒定理**：$\forall$ 不相邻顶点 $u,v$，若 $d(u)+d(v)\ge n$，则 $G$ 是哈密顿图。

## 五、平面图

### 5.1 定义
能画在平面上使边不相交的图。

**面**：边将平面分割成的区域。

**欧拉公式**：$n - m + r = 2$
- n = 顶点数，m = 边数，r = 面数

### 5.2 平面图的必要条件

对于 $n\ge 3$ 的连通简单平面图：
$$m \le 3n - 6$$

**推论**：$K_5$（完全图5）不是平面图，因为 $m=10 > 3\times 5-6 = 9$。

### 5.3 库拉托夫斯基定理
一个图是平面图 $\Leftrightarrow$ 它不包含 $K_5$ 或 $K_{3,3}$ 的细分。

## 六、图的着色

**点着色**：给每个顶点涂色，相邻顶点颜色不同。

**色数 $\chi(G)$**：最少需要的颜色数。

**四色定理**：任何平面图可用至多 4 种颜色着色。
