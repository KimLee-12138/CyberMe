# 组合计数

> 一句话：计数是离散数学的基本功。排列、组合、鸽巢原理、递推关系构成四大核心。

## 一、基本计数原理

### 1.1 加法原理
若做一件事有 $n$ 类方法，第 $i$ 类有 $m_i$ 种，则总共 $\sum m_i$ 种。

### 1.2 乘法原理
若做一件事分 $n$ 步，第 $i$ 步有 $m_i$ 种选择，则共 $\prod m_i$ 种。

### 1.3 包含排斥原理

$$|A_1\cup A_2\cup...\cup A_n| = \sum_{i=1}^n |A_i| - \sum_{i<j}|A_i\cap A_j| + \sum_{i<j<k}|A_i\cap A_j\cap A_k| - ... + (-1)^{n-1}|A_1\cap...\cap A_n|$$

## 二、排列与组合

### 2.1 基本公式

| 类型 | 公式 | 说明 |
|------|------|------|
| **排列 P(n,r)** | $\frac{n!}{(n-r)!}$ | 有序选取 r 个 |
| **组合 C(n,r)** | $\frac{n!}{r!(n-r)!}$ | 无序选取 r 个 |
| **全排列** | $n!$ | n 个元素全排列 |
| **重复排列** | $n^r$ | 有放回选取 |
| **重复组合** | $\binom{n+r-1}{r}$ | 有放回+无序 |

### 2.2 组合恒等式

- $\binom{n}{k} = \binom{n}{n-k}$
- $\binom{n}{k} = \binom{n-1}{k} + \binom{n-1}{k-1}$（**杨辉三角/帕斯卡恒等式**）
- $\sum_{k=0}^n \binom{n}{k} = 2^n$

### 2.3 二项式定理

$$(x+y)^n = \sum_{k=0}^n \binom{n}{k}x^k y^{n-k}$$

## 三、鸽巢原理

### 3.1 基本形式
若把 $n+1$ 个物体放入 $n$ 个盒子，至少有一个盒子含 $\ge 2$ 个物体。

### 3.2 推广形式
若把 $N$ 个物体放入 $k$ 个盒子，至少有一个盒子含 $\ge \lceil N/k \rceil$ 个物体。

### 3.3 应用
- 证明：任意 13 人中至少 2 人在同月出生
- 证明：在 6 个人中必有 3 人互相认识或 3 人互不认识（拉姆齐数 R(3,3)=6）

## 四、递推关系

### 4.1 常系数线性递推

$a_n = c_1a_{n-1} + c_2a_{n-2} + ... + c_ka_{n-k}$

**解法**：特征方程 $r^k - c_1r^{k-1} - ... - c_k = 0$

### 4.2 经典递推

| 序列 | 递推 | 解 |
|------|------|-----|
| 斐波那契 | $F_n = F_{n-1}+F_{n-2}$ | $F_n = \frac{1}{\sqrt{5}}(\varphi^n-\psi^n)$ |
| 卡特兰数 | $C_{n+1}=\sum_{i=0}^n C_iC_{n-i}$ | $C_n = \frac{1}{n+1}\binom{2n}{n}$ |
| 汉诺塔 | $T_n = 2T_{n-1}+1$ | $T_n = 2^n-1$ |

## 五、生成函数

用幂级数表示数列：
$$G(x) = \sum_{n=0}^{\infty} a_n x^n$$

**应用**：求解递推关系、组合计数。
