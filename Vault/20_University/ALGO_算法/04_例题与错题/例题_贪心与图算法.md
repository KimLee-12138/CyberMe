## 贪心篇

### 题 1. 分数背包 ★★☆

容量 50，物品 $(10,60), (20,100), (30,120)$。求最大价值。

**解**：按单位价值排序，依次全取或部分取。总价值 $= 60+100+80 = 240$。

### 题 2. 分发饼干 ★☆☆ [LC 455]

每个孩子有胃口值，每块饼干有尺寸。求最多能满足的孩子数。

**解**：排序后贪心——用最小的能满足的饼干去满足胃口最小的孩子。

```
int findContentChildren(vector<int>& g, vector<int>& s) {
    sort(g.begin(), g.end()); sort(s.begin(), s.end());
    int i = 0, j = 0;
    while (i < g.size() && j < s.size()) {
        if (s[j] >= g[i]) i++;
        j++;
    }
    return i;
}
```

### 题 3. 无重叠区间 ★★☆ [LC 435]

移除最少区间使剩余区间不重叠。

**解**：按**结束时间**排序，每次选最早结束且不与已选区间重叠的区间。贪心正确性：选最早结束的能留给后面最多空间。

```
int eraseOverlapIntervals(vector<vector<int>>& intervals) {
    sort(intervals.begin(), intervals.end(),
         [](auto& a, auto& b) { return a[1] < b[1]; });
    int cnt = 0, last_end = INT_MIN;
    for (auto& iv : intervals) {
        if (iv[0] >= last_end) last_end = iv[1];
        else cnt++;
    }
    return cnt;
}
```

### 题 4. 跳跃游戏 ★★☆ [LC 55]

数组每个元素表示从该位置能跳的最远距离。判断能否到终点。

**解**：贪心维护当前能到达的最远位置。若当前位置 > 最远可达，返回 false。

```
bool canJump(vector<int>& A) {
    int farthest = 0;
    for (int i = 0; i < A.size(); i++) {
        if (i > farthest) return false;
        farthest = max(farthest, i + A[i]);
    }
    return true;
}
```

### 题 5. 跳跃游戏 II ★★★ [LC 45]

求跳到终点的最少步数。

**解**：BFS/贪心——每一步跳跃到当前可达范围内能跳到最远的位置。

```
int jump(vector<int>& A) {
    int steps = 0, curEnd = 0, farthest = 0;
    for (int i = 0; i < A.size() - 1; i++) {
        farthest = max(farthest, i + A[i]);
        if (i == curEnd) { steps++; curEnd = farthest; }
    }
    return steps;
}
```

$O(n)$。

### 题 6. 加油站 ★★★ [LC 134]

环形路线上选起点，保证能绕一圈。

**解**：若总油量 ≥ 总消耗，则必存在起点。从 0 出发，一旦油量变负，将起点重置为下一站。

```
int canCompleteCircuit(vector<int>& gas, vector<int>& cost) {
    int total = 0, tank = 0, start = 0;
    for (int i = 0; i < gas.size(); i++) {
        total += gas[i] - cost[i];
        tank += gas[i] - cost[i];
        if (tank < 0) { start = i + 1; tank = 0; }
    }
    return total >= 0 ? start : -1;
}
```

## 图算法篇

### 题 8. 岛屿数量 ★★☆ [LC 200]

二维网格中 '1' 是陆地，'0' 是水，求岛屿数量。

**解**：DFS/BFS 遍历每个陆地，将连通的陆地标记为已访问。

```
void dfs(vector<vector<char>>& grid, int i, int j) {
    if (i<0 || j<0 || i>=grid.size() || j>=grid[0].size() || grid[i][j]=='0') return;
    grid[i][j] = '0';
    dfs(grid, i+1, j); dfs(grid, i-1, j);
    dfs(grid, i, j+1); dfs(grid, i, j-1);
}
```

$O(mn)$。

### 题 9. 课程表 ★★☆ [LC 207]

有 $n$ 门课和先修关系，判断能否完成所有课程。

**解**：拓扑排序（BFS/Kahn）。若最终输出的课程数 < $n$，则存在环。

```
bool canFinish(int n, vector<vector<int>>& prerequisites) {
    vector<vector<int>> adj(n);
    vector<int> indegree(n, 0);
    for (auto& p : prerequisites) { adj[p[1]].push_back(p[0]); indegree[p[0]]++; }
    queue<int> q;
    for (int i = 0; i < n; i++) if (indegree[i] == 0) q.push(i);
    int cnt = 0;
    while (!q.empty()) {
        int u = q.front(); q.pop(); cnt++;
        for (int v : adj[u]) if (--indegree[v] == 0) q.push(v);
    }
    return cnt == n;
}
```

### 题 10. 网络延迟时间 ★★★ [LC 743]

从源节点 $k$ 出发，信号到所有节点的最短时间。

**解**：Dijkstra 求单源最短路径，取 $\max(dist)$。

```
int networkDelayTime(vector<vector<int>>& times, int n, int k) {
    vector<vector<pair<int,int>>> adj(n+1);
    for (auto& t : times) adj[t[0]].push_back({t[1], t[2]});
    vector<int> dist(n+1, INT_MAX);
    priority_queue<pair<int,int>, vector<pair<int,int>>, greater<>> pq;
    dist[k] = 0; pq.push({0, k});
    while (!pq.empty()) {
        auto [d, u] = pq.top(); pq.pop();
        if (d > dist[u]) continue;
        for (auto [v, w] : adj[u])
            if (dist[u] + w < dist[v]) { dist[v] = dist[u] + w; pq.push({dist[v], v}); }
    }
    int ans = *max_element(dist.begin()+1, dist.end());
    return ans == INT_MAX ? -1 : ans;
}
```

### 题 14. 冗余连接 ★★☆ [LC 684]

树加了一条额外的边，找可删除的边使图变回树。

**解**：并查集。遍历边，若两端点已经连通（同集合），则此边为冗余。

### 题 15. Dijkstra vs Floyd 对比题 ★★★

$V=100$ 的图，需要查询 $Q=10^5$ 次任意两点间的最短距离。应选哪个算法？

**解**：Floyd 一次 $O(V^3)$ 预处理 + $O(1)$ 查询 = $10^6$。Dijkstra 每次查询 $O((V+E)\log V)$，$10^5$ 次查询 = $10^7$ 级别。选 **Floyd**。

### 题 17. N 皇后 ★★★ [LC 51]

在 $n\times n$ 棋盘放 $n$ 个皇后，返回所有摆法。

**解**：逐行放置 + 列/对角线检查回溯。

```
void solve(int row, int n, vector<int>& cols, vector<vector<string>>& ans) {
    if (row == n) { /* 生成棋盘加入 ans */ return; }
    for (int col = 0; col < n; col++) {
        if (isSafe(cols, row, col)) {
            cols.push_back(col);
            solve(row + 1, n, cols, ans);
            cols.pop_back();
        }
    }
}
```

### 题 19. 子集 ★★☆ [LC 78]

求所有子集（幂集）。

**解**：
- **回溯**：每个元素选/不选。
- **迭代**：从空集开始，逐个加入新元素。

```
vector<vector<int>> subsets(vector<int>& A) {
    vector<vector<int>> ans = {{}};
    for (int x : A) {
        int sz = ans.size();
        for (int i = 0; i < sz; i++) {
            ans.push_back(ans[i]);
            ans.back().push_back(x);
        }
    }
    return ans;
}
```

### 题 21. 括号生成 ★★☆ [LC 22]

生成 $n$ 对括号的所有有效组合。

**解**：回溯，约束条件——左括号数 ≥ 右括号数，且左括号数 ≤ $n$。

```
void backtrack(int open, int close, int n, string& s, vector<string>& ans) {
    if (s.size() == 2*n) { ans.push_back(s); return; }
    if (open < n) { s.push_back('('); backtrack(open+1, close, n, s, ans); s.pop_back(); }
    if (close < open) { s.push_back(')'); backtrack(open, close+1, n, s, ans); s.pop_back(); }
}
```
