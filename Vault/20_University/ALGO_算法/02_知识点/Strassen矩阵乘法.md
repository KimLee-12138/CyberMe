# Strassen 矩阵乘法

> 一句话：通过巧妙的代数变换，将 $n \times n$ 矩阵乘法的复杂度从 $O(n^3)$ 降到 $O(n^{2.81})$。

## 一、问题背景

两个 $n \times n$ 矩阵 $A$ 和 $B$ 相乘得 $C = A \times B$：

$$C_{ij} = \sum_{k=1}^n A_{ik} \cdot B_{kj}$$

传统三重循环需要 $n^3$ 次乘法，$O(n^3)$。

## 二、朴素分治法（无改进）

将矩阵各切成 4 个 $n/2 \times n/2$ 子矩阵：

$$A = \begin{bmatrix} A_{11} & A_{12} \\ A_{21} & A_{22} \end{bmatrix}, \quad B = \begin{bmatrix} B_{11} & B_{12} \\ B_{21} & B_{22} \end{bmatrix}$$

根据乘法规则，$C$ 的 4 个分块各需 2 次子矩阵乘法：

- $C_{11} = A_{11}B_{11} + A_{12}B_{21}$
- $C_{12} = A_{11}B_{12} + A_{12}B_{22}$
- $C_{21} = A_{21}B_{11} + A_{22}B_{21}$
- $C_{22} = A_{21}B_{12} + A_{22}B_{22}$

共 **8 次**乘法 + 4 次加法。递归方程：

$$T(n) = 8T(n/2) + O(n^2)$$

主定理：$\log_2 8 = 3$，$f(n)=O(n^2)$ 更低阶(case 1)：**$T(n)=O(n^3)$，无改进**。

## 三、Strassen 的突破

Strassen 发现只需 **7 次**乘法（增加一些加减法）即可算出结果。

### 3.1 7 个中间矩阵 $M_1 \sim M_7$

- $M_1 = (A_{11} + A_{22})(B_{11} + B_{22})$
- $M_2 = (A_{21} + A_{22}) \cdot B_{11}$
- $M_3 = A_{11} \cdot (B_{12} - B_{22})$
- $M_4 = A_{22} \cdot (B_{21} - B_{11})$
- $M_5 = (A_{11} + A_{12}) \cdot B_{22}$
- $M_6 = (A_{21} - A_{11})(B_{11} + B_{12})$
- $M_7 = (A_{12} - A_{22})(B_{21} + B_{22})$

### 3.2 通过加减法组合

- $C_{11} = M_1 + M_4 - M_5 + M_7$
- $C_{12} = M_3 + M_5$
- $C_{21} = M_2 + M_4$
- $C_{22} = M_1 - M_2 + M_3 + M_6$

### 3.3 伪代码

```
Function STRASSEN(A, B, n)
    if n == 1 then return A * B

    将 A, B 分成 n/2 × n/2 子矩阵 A11..B22

    M1 = STRASSEN(A11 + A22, B11 + B22, n/2)
    M2 = STRASSEN(A21 + A22, B11, n/2)
    M3 = STRASSEN(A11, B12 - B22, n/2)
    M4 = STRASSEN(A22, B21 - B11, n/2)
    M5 = STRASSEN(A11 + A12, B22, n/2)
    M6 = STRASSEN(A21 - A11, B11 + B12, n/2)
    M7 = STRASSEN(A12 - A22, B21 + B22, n/2)

    C11 = M1 + M4 - M5 + M7
    C12 = M3 + M5
    C21 = M2 + M4
    C22 = M1 - M2 + M3 + M6

    return C
```

## 四、复杂度分析

递归方程：$$T(n) = 7T(n/2) + O(n^2)$$

主定理：$a=7, b=2$，$\log_2 7 \approx 2.807$

$f(n) = O(n^2)$ 比 $n^{2.807}$ 低了一个 $n^{0.807}$ 因子 → **Case 1**：

$$T(n) = \Theta(n^{\log_2 7}) \approx O(n^{2.807})$$

这比 $O(n^3)$ 有本质改进。

## 五、实用性与局限性

**优势**：渐进复杂度更低

**局限**：
1. **常数因子大**：加减法次数远超传统方法，在矩阵较小时反而慢
2. **数值稳定性差**：多次加减可能导致精度损失
3. **递归开销**：子矩阵切分和合并有额外开销

实际应用中，当 $n$ 很大（通常 $n > 100$）时 Strassen 才开始有优势。

## 六、后续发展

- Coppersmith-Winograd 算法：$O(n^{2.376})$（常数极大，无实用价值）
- 目前理论最优：$O(n^{2.37286})$（仍非实用算法）
- 实践中常用 **分块矩阵乘法 (Blocked Matrix Multiply)**，利用 CPU 缓存局部性

## 七、易错点

1. **Strassen 公式不要求背诵**，但要理解"7 次替代 8 次"的思想
2. 主定理 Case 1 判断：$\log_2 7 > 2$，$f(n)=O(n^2)$ 较低阶
3. **不是所有分治都能降复杂度**：朴素 8 次分没有改进
4. 加减法 $O(n^2)$ 被淹没于递归乘法中，不影响主定理结果
