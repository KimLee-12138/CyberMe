# 高级动态规划技巧

> 一句话：DP 不止于填二维表。状态压缩、滚动数组、树形 DP、背包变形是进阶必经之路。

## 一、滚动数组优化（空间降维）

### 1.1 原理

当 DP 状态转移只依赖上一行（或上一阶段）时，只需保留当前行和上一行。

**例（LCS）**：$dp[i][j]$ 只依赖 $dp[i-1][j-1]$、$dp[i-1][j]$、$dp[i][j-1]$，可降为 $O(\min(m,n))$ 空间。

```
// 0-1 背包：逆序遍历降为一维
vector<int> dp(C+1, 0);
for (int i = 0; i < n; i++)
    for (int j = C; j >= w[i]; j--)   // 逆序！
        dp[j] = max(dp[j], dp[j-w[i]] + v[i]);
```

**为什么 0-1 背包必须逆序**：正序会导致同一物品被多次选用（变成完全背包）。

### 1.2 完全背包（正向遍历）

```
for (int j = w[i]; j <= C; j++)   // 正序！
    dp[j] = max(dp[j], dp[j-w[i]] + v[i]);
```

## 二、状态压缩 DP

### 2.1 原理

当问题涉及"集合中的元素是否被选"时，用二进制位表示每个元素的选择状态。适合 $n \le 20$ 的 NP 问题。

**旅行商问题 (TSP)**：$dp[mask][i]$ = 已访问城市集合为 mask、当前在城市 $i$ 的最短路径。

```
// 状态转移：从 i 走到 j（j 未被访问）
dp[mask | (1<<j)][j] = min(dp[mask | (1<<j)][j], dp[mask][i] + dist[i][j])
```

复杂度 $O(n^2 \cdot 2^n)$。

### 2.2 经典应用

- 集合覆盖
- 哈密顿路径/回路
- 分配问题（n 个人分配 n 个任务的最小代价）[LC 1947]

### 2.3 技巧

- `mask & (1<<i)` 检查第 $i$ 位是否为 1
- `mask | (1<<i)` 将第 $i$ 位设为 1
- `mask ^ (1<<i)` 翻转第 $i$ 位
- 枚举子集：`for (int sub = mask; sub; sub = (sub-1) & mask)`

## 三、树形 DP

### 3.1 原理

在树的结构上做 DP，自底向上（后序遍历）合并子树信息。

**打家劫舍 III** [LC 337]：二叉树中不能偷直接相连的节点。

```
// 返回 {不偷当前节点的最大值, 偷当前节点的最大值}
pair<int,int> dfs(TreeNode* root) {
    if (!root) return {0, 0};
    auto [l_no, l_yes] = dfs(root->left);
    auto [r_no, r_yes] = dfs(root->right);
    int no = max(l_no, l_yes) + max(r_no, r_yes);   // 不偷：子节点可偷可不偷
    int yes = root->val + l_no + r_no;               // 偷：子节点不能偷
    return {no, yes};
}
```

### 3.2 树的直径

**两次 BFS**：任一点 BFS 到最远点 $a$，再从 $a$ BFS 到最远点 $b$，$a-b$ 路径即为直径。

**树形 DP 解**：每个节点维护向下走的最长距离。直径 = $\max($经过当前节点的最长路径$)$。

### 3.3 经典问题

- 树的最大独立集
- 树的最小顶点覆盖
- 树的重心

## 四、背包问题全家桶

| 类型 | 特征 | 内层循环 | 状态转移 |
|------|------|---------|---------|
| 0-1 背包 | 每件 1 个 | 逆序 | $dp[j]=\max(dp[j], dp[j-w]+v)$ |
| 完全背包 | 每件无限 | 正序 | 同上公式，正序遍历 |
| 多重背包 | 每件 $k_i$ 个 | 二进制拆分后转 0-1 | — |
| 分组背包 | 每组选 1 个 | 先容量后组内 | $dp[j]=\max_k(dp[j], dp[j-w_k]+v_k)$ |
| 二维费用 | 两维容量 | 两维逆序 | $dp[j][k]$ |

**多重背包二进制优化**：将 $k_i$ 个物品拆成 $\lceil\log_2 k_i\rceil$ 个 0-1 物品（分别代表 1、2、4、8...个）。

## 五、区间 DP

### 5.1 特征

状态定义为一个区间 $[i,j]$，从小区间向大区间递推。

**模板**：
```
for (int len = 2; len <= n; len++)        // 区间长度
    for (int i = 0; i + len - 1 < n; i++) // 左端点
        int j = i + len - 1;              // 右端点
        for (int k = i; k < j; k++)       // 分割点
            dp[i][j] = combine(dp[i][k], dp[k+1][j])
```

**经典题**：
- 矩阵链相乘
- 戳气球 [LC 312]：$dp[i][j]$ = 戳破 $(i,j)$ 内所有气球的最大得分
- 最长回文子序列 [LC 516]：$dp[i][j] = dp[i+1][j-1]+2$（两端相同）

## 六、数位 DP

统计区间 $[l,r]$ 内满足某些数字性质的数的个数。从高位到低位逐位枚举，记录是否受到上界限制。

**模板适用**：求 $[1,n]$ 中不包含数字 4 的数的个数。

```
// pos: 当前位数, tight: 是否受上界约束, 记忆化
int dfs(int pos, bool tight, vector<vector<int>>& memo, string& s)
```

---

## 七、易错点

1. **0-1 背包逆序，完全背包正序**：搞反了会得到错误答案
2. **状态压缩 DP 的 mask 从 0 到 (1<<n)-1**，不要漏掉 0 状态
3. **区间 DP 的 len 从 2 开始**，len=1 是基准情况
4. **树形 DP 通常是后序遍历**，确保子树结果已算好
