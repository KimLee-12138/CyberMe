# 资料导入清单模板

> 使用说明：每行一个文件。`source_id` 格式为 `src-{课程编号}-{序号}`。所有路径保留完整 Windows 绝对路径。

## 字段说明

| 字段 | 说明 | 示例 |
|---|---|---|
| `source_id` | Vault 内稳定唯一来源 ID | `src-db-0001` |
| `原始路径` | 文件在 `学习资料` 中的绝对路径 | `C:\...\学习资料\数据库\课件.pdf` |
| `文件名` | 当前文件名（不含路径） | `第03章SQL.pdf` |
| `类型` | 文件扩展名（小写） | `pdf`、`docx`、`pptx`、`md`、`zip` |
| `大小(KB)` | 文件大小（KB） | `1024` |
| `修改时间` | 文件最后修改时间 | `2025-09-15` |
| `课程/项目` | 所属课程或项目 | `DB`、`数据结构` |
| `主题` | 内容主题关键词 | `SQL`、`函数依赖`、`范式` |
| `重复组` | 哈希重复组编号，唯一文件留空 | `dup-001` |
| `可提取文本` | 是否可直接提取文本 | `yes`、`no`、`partial` |
| `需要OCR` | 是否需要 OCR | `yes`、`no` |
| `含公式` | 是否包含数学公式 | `yes`、`no` |
| `含表格/图片` | 是否包含表格或图片 | `yes`、`no` |
| `敏感性` | 敏感信息风险 | `normal`、`sensitive`、`unknown` |
| `优先级` | 处理优先级 | `P0`、`P1`、`P2`、`P3` |
| `状态` | 处理状态 | `raw`、`extracted`、`cleaned`、`verified`、`deprecated` |
| `备注` | 其他说明 | 疑似旧版、加密、缺页等 |
